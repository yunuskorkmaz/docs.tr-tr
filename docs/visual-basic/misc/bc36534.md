# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>İfade bir ifade ağacına dönüştürülemez

Çoğu ifade ifade ağaçlarına dönüştürülebilir, ancak bazıları olamaz. Bu hata, dönüştürülemeyen bir ifadeyi dönüştürmek için bir girişimde bulunuldu.  
  
 **Hata kimliği:** BC36534
