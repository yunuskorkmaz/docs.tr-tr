# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>' ' Language öğesinde ' ' XML açıklama etiketine \<tag> izin verilmez \<element>

\<tag>' ' ÖĞESINDE XML açıklama etiketine izin verilmez \<element> . XML açıklaması yoksayılacak.  
  
 XML açıklama etiketi, uygulandığı dil öğesi tarafından desteklenmiyor.  
  
 **Hata kimliği:** BC42306  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Desteklenmeyen XML açıklama etiketini kaldırın.
