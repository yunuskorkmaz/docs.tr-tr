# <a name="comma-expected"></a>Virgül bekleniyor
Bir bağımsız değişken listesi iki bağımsız değişkenleri arasında virgül (,) eksik.  
  
 **Hata Kimliği:** BC30196  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm bağımsız değişkenler virgülle emin olun.
