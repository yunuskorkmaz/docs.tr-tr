# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-it-is-declared-39shared39"></a>&#39;&lt;declaration1&gt; &#39; geçersiz kılamaz &#39; &lt;declaration2&gt; &#39; onu bildirildiğinden &#39;paylaşılan&#39;
Aynı ada sahip bir devralınan öğe geçersiz kılmak bir yordam veya özellik bildirimi çalışır, ancak devralınan öğesi olarak belirtilen `Shared`. Bunlar kılınamaması paylaşılan öğeleri sınıfın örneklerini ile ilişkili değildir.  
  
 **Hata Kimliği:** BC30268  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Shared` anahtar sözcüğü devralınan öğesi ya da kaldırma geçersiz kılma bildirimi.
