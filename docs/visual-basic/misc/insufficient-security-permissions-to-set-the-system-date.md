# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Sistem tarihini ayarlamak için güvenlik izinleri yetersiz
Güvenlik izinlerini sistem tarihi ayarlamak yeterli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sistem tarihi ayarını izin vermek için güvenlik izinlerini değiştirir.
