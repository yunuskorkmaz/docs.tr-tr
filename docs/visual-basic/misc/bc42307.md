# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML açıklama parametresi &#39; &lt;parametresi&gt; &#39; üzerinde karşılık gelen bir parametre eşleşmiyor &lt;anahtar sözcüğü&gt; deyimi
XML açıklama parametresi \<parametresi > üzerinde karşılık gelen bir parametre eşleşmiyor \<anahtar sözcüğü > deyimi. XML açıklama yoksayılacak.  
  
 Bir XML açıklama etiketi, ancak tüm parametreler için de bir yöntem belirtildi.  
  
 **Hata Kimliği:** BC42307  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML açıklama etiketi tüm parametreler için de yöntemi sağlar.
