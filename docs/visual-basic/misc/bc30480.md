# <a name="shared-sub-new-cannot-be-declared-specifier"></a>Paylaşılan ' Sub New ', ' ' olarak bildirilemez \<specifier>
Bildirimde geçerli olmayan bir tanımlayıcı kullanmayı denediniz `Shared Sub New` .  
  
 **Hata kimliği:** BC30480  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Belirteci kaldırın.
