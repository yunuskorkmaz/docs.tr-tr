# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>Dizi boyutları ' VBFixedArray ' özniteliğinde belirtilenler ile eşleşmiyor
Dizinizdeki Boyutlar, özniteliğinde belirttiğiniz boyutlarla eşleşmez `VBFixedArray` .  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Dizi deyimindeki dizi boyutlarının `Dim` öznitelik tarafından belirtilen boyutlarla eşleştiğinden emin olun `VBFixedArray` .
