# <a name="instance-members-and-me-cannot-be-used-within-a-lambda-expression-in-structures"></a>Örnek üyeleri ve ' Me ' yapılardaki bir lambda ifadesi içinde kullanılamaz
Bir yapının içinden, yapının örnek üyesine başvuran bir lambda ifadesi tanımladınız veya kullanır `Me` . Aşağıdaki kod, bu iki geçersiz başvuruyu gösterir.  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```  
  
 **Hata kimliği:** BC36638  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Örnek üyesini yerel bir değişkene atayın ve deyiminizdeki yerel değişkeni kullanın.  
  
```vb  
Public Function ExampleFunFix() As Integer  
    Dim temp = InstanceMember  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
