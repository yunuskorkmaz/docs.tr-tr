# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>En fazla hata sayısı aşıldı
En fazla 100 hata derleyiciye ulaşıldı.  
  
 **Hata kimliği:** BC2020  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Hata ayıklama ve her sırayla adresleme ile oluşturulan hata sayısını azaltın.
