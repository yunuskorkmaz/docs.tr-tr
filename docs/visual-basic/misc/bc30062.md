# <a name="39me39-cannot-be-the-target-of-an-assignment"></a>&#39;Bana&#39; atama işleminin hedefi olamaz
`Me` Anahtar sözcüğü her zaman burada kod şu anda yürütülmekte olan bir sınıfın belirli örneğine başvurur. Değeri değiştirilemez.  
  
 **Hata Kimliği:** BC30062  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir değer atayan deyimi kaldırmak `Me`.
