# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; gövdesi boş olmayan bir alt, işlev veya işleci uygulanamaz
`DllImportAttribute` Özniteliği için uygulandığı bir `Sub`, `Function`, veya `Operator` boş değil.  
  
 **Hata Kimliği:** BC31522  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm koddan kaldırmak `Sub`, `Function`, veya `Operator` bu öznitelik kullanılacak.
