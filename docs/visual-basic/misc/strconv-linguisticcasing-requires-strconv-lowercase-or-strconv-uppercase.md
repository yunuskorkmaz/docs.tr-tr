# <a name="39strconvlinguisticcasing39-requires-39strconvlowercase39-or-39strconvuppercase39"></a>&#39; StrConv.LinguisticCasing &#39; gerektirir &#39; StrConv.LowerCase &#39; ya &#39; StrConv.UpperCase &#39;
Kullanma girişiminde bulundunuz `StrConv.LinguisticCasing`, olduğu yalnızca birlikte geçerli `StrConv.LowerCase` veya `StrConv.UpperCase`.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kullanım `StrConv.LowerCase` veya `StrConv.UpperCase` birlikte `StrConv.LinguisticCasing`.
