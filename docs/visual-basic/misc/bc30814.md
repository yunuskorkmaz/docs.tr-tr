# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39; GoSub &#39; deyimleri artık desteklenmiyor
`GoSub`bir yordam çağırma için kullanılamaz.  
  
 **Hata Kimliği:** BC30814  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yordamları kullanmadan doğrudan çağırmak `GoSub`; örneğin:  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
