# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39; bildirme &#39; bir yapı deyimlerinde bildirilemez &#39; &lt;belirticisi&gt;&#39;
A `Declare` bildirime sahip içinde geçersiz bir belirleyici bir `Structure` bildirimi.  
  
 **Hata Kimliği:** BC30791  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
