# <a name="3939-expected-declaration"></a>&#39;=&#39; (bildirim) bekleniyor
A `Mid` deyimi veya `#Const` derleme yönergesi atama değeri içermiyor.  
  
 **Hata Kimliği:** BC30249  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Eşittir işareti ekleyin (`=`) ve dize ya da derleme sabiti atanacak değer.
