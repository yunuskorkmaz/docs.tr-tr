# <a name="collection-index-must-be-in-the-range-1-to-the-size-of-the-collection"></a>Koleksiyon dizin koleksiyon boyutu 1 aralığında olmalıdır
Sağlanan dizin değeri 1 ' toplama boyutu için geçerli aralık dışında döner.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Sağlanan değer geçerli bir değiştirin.
