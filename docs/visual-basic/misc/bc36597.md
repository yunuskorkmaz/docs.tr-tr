# <a name="goto-to-label-labelname-cannot-be-used-within-lambda-expressions"></a>' ' Etiketine git \<labelname> Lambda ifadelerinde kullanılamaz
Lambda ifadesinin gövdesi, tek bir değer hesaplayan ve döndüren bir ifade olmalıdır. Deyimler dahil hiçbir deyime izin verilmez `GoTo` .  
  
 **Hata kimliği:** BC36597
