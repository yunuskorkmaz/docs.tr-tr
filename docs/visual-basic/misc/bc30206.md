# <a name="option-must-be-followed-by-compare-explicit-infer-or-strict"></a>' Option ' arkasından ' Compare ', ' Explicit ', ' Infer ' veya ' Strict ' gelmelidir
İfadesiyle ayarlayabileceğiniz tek Seçenekler `Option` ,, ve ' dir `Compare` `Explicit` `Infer` `Strict` .  
  
 **Hata kimliği:** BC30206  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bir seçenek anahtar sözcüğünün ( `Compare` , `Explicit` , `Infer` veya `Strict` ) yanlış yazılmış olup olmadığını kontrol edin.  
  
- Deyimin tamamlanmamış olup olmadığını kontrol edin `Option` .
