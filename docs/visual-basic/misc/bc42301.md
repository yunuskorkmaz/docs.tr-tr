# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Language öğesi yalnızca bir XML açıklama bloğu izin verilir
Birden çok XML açıklama bloğu bir dil öğesine uygulanmıştır.  
  
 **Hata Kimliği:** BC42301  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz XML açıklama bloğunu kaldırın.
