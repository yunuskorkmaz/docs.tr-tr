# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>Türü &#39; &lt;typename&gt;&#39; derleme &#39; farklı sürümlerinden alınan&lt; AssemblyName&gt;&#39;
Türü '\<typename >' derlemenin farklı sürümleri alınan '\<assemblyname >'. Aynı derlemenin farklı sürümleri aynı projede kullanılamaz.  
  
 Bir tür aynı derlemenin belirsizlik oluşturma farklı sürümlerini içeri aktarılmış.  
  
 **Hata Kimliği:** BC31525  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok sürümü aynı derleme başvuruları kaldırın.
