# <a name="attribute-value-is-not-valid-expecting-39value139-or-39value239"></a>Öznitelik değeri geçerli değil; bekleniyor &#39;value1&#39; veya &#39;value2&#39;
Bir XML belgesi değişmez değer özniteliği için geçersiz değer belirtildi. Bir XML belgesi değişmez değeri geçerli öznitelik değerlerini XML 1.0 belirtimi temel alır.  
  
 **Hata Kimliği:** BC31182  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML 1.0 belirtimi geçerli bir değer için öznitelik değerini değiştirin.
