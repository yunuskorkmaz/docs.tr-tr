# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Öznitelik değeri geçerli değil; ' değer1 ' veya ' değer2 ' bekleniyor
XML belgesi sabit değerinde bir öznitelik için geçersiz bir değer belirtildi. XML belgesi değişmez değeri için geçerli öznitelik değerleri, XML 1,0 belirtimine dayanır.  
  
 **Hata kimliği:** BC31182  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öznitelik değerini XML 1,0 belirtimine göre geçerli olan bir değer olarak değiştirin.
