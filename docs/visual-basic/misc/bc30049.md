# <a name="statementname-statement-requires-an-array"></a>' \<statementname> ' ifadesiyle bir dizi gerekiyor
`Erase`Ve `ReDim` deyimleri, dizi veri türleri gerektirir. Sağlanan tür bir dizi değil.  
  
 **Hata kimliği:** BC30049  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Veri türünü bir dizi türü olarak değiştirin.
