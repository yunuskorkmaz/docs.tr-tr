# <a name="xml-namespace-prefix-prefix-is-not-defined"></a>' Önek ' XML ad alanı öneki tanımlanmadı
XML sabit değeri veya XML ekseni özelliği için tanımsız bir XML ad alanı öneki belirtildi.  
  
 **Hata kimliği:** BC31148  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- İfadesini kullanarak XML ad alanı önekini tanımlayın `Imports` .  
  
 -veya-  
  
- XML sabit değeri veya XML eksen özelliği için XML ad alanı önekini önceden tanımlanmış bir XML ad alanı ön eki olarak değiştirin.
