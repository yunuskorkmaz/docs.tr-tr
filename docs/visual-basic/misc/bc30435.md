# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>Yapı üyeleri bildirilemez &#39; &lt;belirticisi&gt;&#39;
İçinde bir üyede geçerli olmayan bir belirleyici kullanılan bir `Structure` deyimi.  
  
 **Hata Kimliği:** BC30435  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
