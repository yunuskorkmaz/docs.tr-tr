# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>Üye &#39; &lt;membername&gt; &#39; sınıfında bulunamıyor &#39; &lt;classname&gt;&#39;
Üye '\<membername >' sınıfında bulunamıyor\<classname >'. Bu durum genellikle bir eşleşmeyen 'Microsoft.VisualBasic.dll içinde' sonucudur.  
  
 Tanımlanan üyesi bulunamadı.  
  
 **Hata Kimliği:** BC31097  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Hata durumunda yinelenen yeniden görmek için programı derleyin.  
  
2.  Hata tekrar oluşursa, çalışmanızı kaydedin ve Visual Studio'yu yeniden başlatın.  
  
3.  Sorun devam ederse, Visual Basic yeniden yükleyin.  
  
4.  Yeniden yüklenmesinden sonra hata devam ederse, Microsoft Ürün Destek Hizmetleri'ne bildirin.
