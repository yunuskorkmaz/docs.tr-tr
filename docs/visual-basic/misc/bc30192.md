# <a name="end-of-parameter-list-expected"></a>Beklenen parametre listesinin sonuna
A `ParamArray` belirtimi bir veya daha fazla diğer bağımsız değişkeni belirtimleri yordamı bildiriminde ardında.  
  
 **Hata Kimliği:** BC30192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `ParamArray` sonuna kadar bağımsız değişken listesi.
