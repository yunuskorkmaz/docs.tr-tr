# <a name="declare-statements-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Bir modül deyimlerinde bildirilemez bildirme &#39; &lt;belirticisi&gt;&#39;
A `Declare` bildirime sahip içinde geçersiz bir belirleyici bir `Module` bildirimi. Modüller hiçbir zaman oluşturulabilir, devralma desteklemez ve arabirimleri uygulayamaz.  
  
 **Hata Kimliği:** BC30786  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
