# <a name="object-initializer-syntax-cannot-be-used-to-initialize-an-instance-of-type-object"></a>Nesne başlatıcısı sözdizimi, ' Object ' türünde bir örneği başlatmak için kullanılamaz
Nesne Başlatıcısı söz dizimini kullanarak bir `Object` örneğini başlatıyükleyemezsiniz. Bir `Object` örneğinin bir değer atamak için hiçbir özelliği veya alanı yoktur ve nesne Başlatıcısı söz dizimi en az bir tür özelliği veya alanı gerektirir.  
  
```vb  
' Not valid.  
' Dim obj1 = New Object With {}  
' Dim obj2 = New Object With {.ToString = <some value>}  
```  
  
 **Hata kimliği:** BC30994  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1. Başlatıcı listesi kullanmadan `Object` türünde örnekler bildirin:  
  
    ```vb  
    Dim obj3 as Object  
    Dim obj4 as New Object()  
    ```
