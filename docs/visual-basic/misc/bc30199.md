# <a name="-expected"></a>' (' bekleniyor
Yordam çağrısındaki bir bağımsız değişken listesi, açma ayracı ( `(` ) karakteriyle başlamamalıdır.  
  
 **Hata kimliği:** BC30199  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Listenin başlangıcına bir açma parantezi karakteri ekleyin.
