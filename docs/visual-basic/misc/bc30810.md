# <a name="methods-cannot-be-declared-static"></a>Metotlar ' static ' olarak bildirilemez

 **Hata kimliği:** BC30810  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- `Static`Bir yordamın içindeki bağımsız öğeleri statik olarak bildirmek için değiştiricisini kullanın.
