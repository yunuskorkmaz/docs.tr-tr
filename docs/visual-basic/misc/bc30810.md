# <a name="methods-cannot-be-declared-39static39"></a>Yöntemleri bildirilemez &#39; Statik &#39;
`Static` Değiştiricisi tüm yordamları uygulanamaz.  
  
 **Hata Kimliği:** BC30810  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Static` statik olarak bir yordam içindeki ayrı öğeleri bildirmek için değiştiricisi.
