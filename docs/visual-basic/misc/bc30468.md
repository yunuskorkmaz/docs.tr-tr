# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>Tür bildirim karakterleri bu bağlamda geçerli değil

 **Hata Kimliği:** BC30468  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Karakter kaldırın.
