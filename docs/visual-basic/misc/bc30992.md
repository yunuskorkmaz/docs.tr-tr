## <a name="example"></a><span data-ttu-id="2faf3-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="2faf3-109">Example</span></span>  

 <span data-ttu-id="2faf3-110">Aşağıdaki sınıf, `defaultProp` bir tamsayı bağımsız değişkeni gerektiren varsayılan bir özelliğini içerir.</span><span class="sxs-lookup"><span data-stu-id="2faf3-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="2faf3-111">Varsayılan özellik bir bağımsız değişken gerektirdiğinden, aşağıdaki bildirim bir hataya neden olur.</span><span class="sxs-lookup"><span data-stu-id="2faf3-111">Because the default property requires an argument, the following declaration causes an error.</span></span>

