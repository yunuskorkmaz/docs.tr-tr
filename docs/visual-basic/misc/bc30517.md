# <a name="overload-resolution-failed-because-no-39ltmethodgt39-is-accessible"></a>Aşırı yükleme çözümü başarısız oldu, çünkü yok &#39; &lt;yöntemi&gt;&#39; erişilebilir
Aşırı yüklenmiş yöntemine bir çağrı yapılır ancak derleyici parametrelerin sayısı bakımından eşleşen bir parametre listesi erişilebilir bir yöntemle bulamıyor.  
  
 **Hata Kimliği:** BC30517  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Onay, sağladığınız bağımsız değişken listesi.  
  
2.  Erişilebilir olup olmadığını belirlemek için aranacak çalıştığınız yöntemi denetleyin.
