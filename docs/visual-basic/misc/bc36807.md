# <a name="xml-elements-cannot-be-selected-from-type-39type39"></a>XML öğeleri türü &#39; türü &#39;seçilemez;
Bir XML alt öğe türü değil bir nesne için başvurulan <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, veya `IEnumerable(Of XElement)`. Daha fazla bilgi için bkz: [XML alt Axis özelliği](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).  
  
```vb  
' Generates an error.  
Dim var = "sample text".<child>  
```  
  
 **Hata Kimliği:** BC36807  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Hangi, başvuran bir öznitelik nesnesi olarak kesinlikle yazıldığından emin olun <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, veya `IEnumerable(Of XElement)`. Aşağıda bir örnek verilmiştir:  
  
```vb  
Dim elem As XElement = <root>  
                         <child />  
                       </root>  
Dim var = elem.<child>  
```
