# <a name="assembly-attribute-attributename-is-not-valid-error"></a>' ' Derleme özniteliği \<attributename> geçerli değil: \<error>

Ortak dil çalışma zamanı bir derlemede özniteliği kabul etmez.

**Hata kimliği:** BC30129

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Özniteliği bildirimden kaldırın veya bağlamı için geçerli olan bir öznitelik ile değiştirin.

2. Alıntı yapılan hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
