# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-39ltmethodnamegt39"></a>Parametresi için bağımsız değişken belirtilmemiş &#39; &lt;parametername&gt; &#39; , &#39; &lt;methodname&gt;&#39;
Gerekli parametre için bir bağımsız değişken sağladınız değil.  
  
 **Hata Kimliği:** BC30455  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtilen parametre için bir bağımsız değişken sağlayın.
