# <a name="label-labelname-is-not-defined"></a>' \<labelname> ' Etiketi tanımlı değil
Bir çizgi etiketi veya satır numarası, başvurunun kapsamı içinde tanımlanmamış ancak tanımlanmadı. Etiket, başvuruyu içeren yordamın içinde olmalıdır.  
  
 **Hata kimliği:** BC30132  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Satır etiketi ve başvurunun aynı yordamda olması için kodu yeniden yapılandır.
