# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39; Al &#39; deyimi, eşleşen bir &#39;ile bitmelidir; Son Al &#39;
`Get`özellik yordamları ile bitmelidir bir `End Get` deyimi.  
  
 **Hata Kimliği:** BC30631  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Olduğundan emin olun `Get` özellik yordamı sona erer ile bir `End Get` oluşturun.
