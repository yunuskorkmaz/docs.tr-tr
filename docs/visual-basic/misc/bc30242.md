# <a name="modifier-is-not-valid-on-a-method-declaration"></a>' \<modifier> ' yöntem bildiriminde geçerli değil

`Function`Or `Sub` deyimleri geçersiz bir anahtar sözcük içeriyor.  
  
 **Hata kimliği:** BC30242  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Veya deyimindeki geçersiz anahtar sözcüğü kaldırın `Function` `Sub` .
