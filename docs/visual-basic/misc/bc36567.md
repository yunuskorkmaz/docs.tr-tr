# <a name="parameter-parametername-already-has-a-matching-argument"></a>' \<parametername> ' Parametresinin zaten eşleşen bir bağımsız değişkeni var
Bir bağımsız değişken aynı yordam çağrısında bir kereden fazla bir kez geçirilir.  
  
 **Hata kimliği:** BC36567  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Bağımsız değişken adlarından birinin yanlış yazılmış olup olmadığını kontrol edin.  
  
- Bağımsız değişken listesinden yinelenen bağımsız değişkeni kaldırın.
