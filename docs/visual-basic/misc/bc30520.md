# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-from-39lttype1gt39-to-39lttype2gt39"></a>Bağımsız değişken parametre &#39;eşleştirme; &lt;parametername&gt;&#39; öğesinden daraltır &#39;&lt; type1&gt;&#39; çok &#39;&lt; type2&gt;&#39;
Aşırı yüklenmiş yöntemine bir çağrı yapılır, ancak derleyici daraltma dönüşümü olmadan adlı bir yöntemi bulunamıyor. Daraltma dönüşümü tam olarak bazı olası değerleri tutun mümkün olmayabilir veri türü için bir değer değiştirir.  
  
 **Hata Kimliği:** BC30520  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `Option Strict Off`.
