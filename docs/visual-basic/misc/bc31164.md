# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>XML öznitelik değeri için eşleşen kapatma çift tırnak işareti bekleniyordu
Çift tırnak işaretleriyle ayrılmış bir öznitelik değerinde kapanış tırnak işareti karakteri eksik.  
  
 **Hata kimliği:** BC31164  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Öznitelik değerini sınırlandırmak için kapanış çift tırnak işareti karakterini ekleyin.
