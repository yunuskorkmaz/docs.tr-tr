# <a name="end-operator-expected"></a>'Operator End' bekleniyor
Bir `End` deyimi sonlandırmasına blok eşleşmesi gerekir. Bir `Operator` yordam tanımında değil düzgün bir şekilde sonlandırıldı.  
  
 **Hata Kimliği:** BC33005  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme bir `End Operator` yordamı sonuna deyimi.
