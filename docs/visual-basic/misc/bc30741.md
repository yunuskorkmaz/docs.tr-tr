# <a name="typename1-cannot-be-converted-to-typename2"></a>' \<typename1> ', ' ' olarak dönüştürülemez \<typename2>
Uyumsuz iki tür arasında tür dönüştürmesi denendi.  
  
 **Hata kimliği:** BC30741  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Cast olan türleri doğrulayın.
