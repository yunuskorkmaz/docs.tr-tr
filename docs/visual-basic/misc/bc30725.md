# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Throw &#39; deyimleri Komut penceresi içinde geçerli değildir
`Throw`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30725  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Throw` kodda hata ayıklama gelen deyimleri.
