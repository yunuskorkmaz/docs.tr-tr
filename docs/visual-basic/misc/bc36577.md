# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Anonim tür başlatıcıları hata ayıklama Windows 'da geçerli değildir
Bir hata ayıklama oturumu sırasında **Watch** veya **anında** pencereye anonim bir tür başlatıcısı girildi. Anonim tür başlatıcıları Windows hata ayıklama penceresinde kullanılamaz.  
  
 **Hata kimliği:** BC36577  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Kod düzenleyicisinde anonim tür başlatıcıları test edin.
