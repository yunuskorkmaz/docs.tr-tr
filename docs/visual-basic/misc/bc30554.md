# <a name="namespacename-is-ambiguous"></a>' \<namespacename> ' belirsiz
Belirsiz bir ad sağladınız ve bu nedenle başka bir adla çakışıyor. Visual Basic derleyicisinde herhangi bir çakışma çözümü kuralı yok; adları kendiniz kesinleştirmeniz gerekir.  
  
 **Hata kimliği:** BC30554  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Adı tam olarak nitelendirin.
