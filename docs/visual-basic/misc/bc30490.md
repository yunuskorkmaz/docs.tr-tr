# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39;Varsayılan&#39; birleştirilemez &#39; &lt;belirticisi&gt;&#39;
Birleştirme denediniz `Default` bir belirleyici olan anahtar sözcük gibi `Shared` veya `Private`, yani bu bağlamda geçerli değil.  
  
 **Hata Kimliği:** BC30490  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Default`.  
  
     -veya-  
  
-   Çakışan belirleyici kaldırın.
