# <a name="method1-and-method2-cannot-overload-each-other-because-they-differ-only-by-parameters-declared-byref-or-byval"></a>\<method1> \<method2> yalnızca ' ByRef ' veya ' ByVal ' olarak belirtilen parametreleri farklı olduğundan, ' ' ve ' ' birbirini aşırı yükleyemez
Yalnızca veya olarak belirtilen bir parametreyle farklılık gösteren başka bir yöntemle bir yöntemi aşırı yüklemeye çalıştınız `ByRef` `ByVal` .  
  
 **Hata kimliği:** BC30345  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Yöntemlerinin veya parametresinin adından daha fazlasına sahip olduğundan emin olun `ByRef` `ByVal` .
