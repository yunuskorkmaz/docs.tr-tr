# <a name="date-constant-is-not-valid"></a>Tarih sabiti geçerli değil
Bir tarih sabiti geçersiz bir tarih ile bildirilir.  
  
 **Hata Kimliği:** BC31085  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sabitinde kullanılan tarih düzeltin.
