# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>&#39; &lt;typename&gt;&#39; bunu bildirildiğinden öznitelik olarak kullanılamaz &#39; MustInherit &#39;
Özel öznitelik sınıfları olarak bildirilemez `MustInherit`.  
  
 **Hata Kimliği:** BC31506  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `MustInherit` özel öznitelik değiştiricisi.
