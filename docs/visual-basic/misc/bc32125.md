' uygulamasıyla çakışabileceğinden</span><span class="sxs-lookup"><span data-stu-id="3376c-102">Cannot implement '\<interfacename1>.\<membername>' because its implementation could conflict with the implementation for '\<interfacename2>.\<membername>' for some type arguments</span></span>
<span data-ttu-id="3376c-103">Bir sınıf birden çok genel arabirim uygular, bunlardan biri diğerinden devralır ve bir arabirim üyesinin iki uygulaması, tür bağımsız değişkenlerinin belirli değerleri için çakışabilir.</span><span class="sxs-lookup"><span data-stu-id="3376c-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="3376c-104">Aşağıdaki deyimler bu hatayı oluşturabilir.</span><span class="sxs-lookup"><span data-stu-id="3376c-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="3376c-105">`iFace2` `iFace1` , Kendi tür parametresi () kullanmaktan devraldığından `u` , `testClass` `iFace1.testSub` aynı tür bağımsız değişkeni ve ' ye geçirilirse aynı imza ile iki sürümünü uygular `y` `z` .</span><span class="sxs-lookup"><span data-stu-id="3376c-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="3376c-106">Bu, hangi sürümün erişebileceğinizi bir belirsizlik oluşturacak.</span><span class="sxs-lookup"><span data-stu-id="3376c-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="3376c-107">**Hata kimliği:** BC32125</span><span class="sxs-lookup"><span data-stu-id="3376c-107">**Error ID:** BC32125</span></span>
