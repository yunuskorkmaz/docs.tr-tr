# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="5310e-102">Tür parametresi ile bir &#39;yapısı&#39; kısıtlaması kısıtlama olarak kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="5310e-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="5310e-103">Bir tür parametresi ile bir `Structure` kısıtlaması için başka bir tür parametresi kısıtlaması olarak kullanılır.</span><span class="sxs-lookup"><span data-stu-id="5310e-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="5310e-104">`Structure` Kısıtlaması, tür parametresi için geçirilen tür bağımsız değişkeni bir değer türü olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="5310e-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="5310e-105">Ancak, bir değer türü uygulanan veya olamaz devralınan, bu nedenle, uygulamaya veya bu türden devralmalıdır için diğer tür parametresi gerektiren bir kısıtlama olarak kullanılacak anlamlı değildir.</span><span class="sxs-lookup"><span data-stu-id="5310e-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="5310e-106">Bu durum yalnızca anlamlı yorumunu her iki tür bağımsız değişkenleri tamamen aynı türdeki olmanızın gerekmesidir.</span><span class="sxs-lookup"><span data-stu-id="5310e-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="5310e-107">Bu durumda, yalnızca bir tür parametresi, genel tür gerekir.</span><span class="sxs-lookup"><span data-stu-id="5310e-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="5310e-108">Aşağıdaki deyim, bu hata oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="5310e-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="5310e-109">Türü geçirilen `t2` uygulayan veya geçirilen tür devralma `t1`, türü geçirilen `t1` bir değer türü olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="5310e-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="5310e-110">**Hata Kimliği:** BC32114</span><span class="sxs-lookup"><span data-stu-id="5310e-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5310e-111">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="5310e-111">To correct this error</span></span>  
  
-   <span data-ttu-id="5310e-112">Kaldırmak için kısıtlanmış bir tür parametresi `Structure` bir tür parametresi kısıtlaması listesinden.</span><span class="sxs-lookup"><span data-stu-id="5310e-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="5310e-113">Her iki tür parametreleri aynı değer türü gerektiriyorsa, yalnızca bir tür parametresi ile genel tür tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="5310e-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
