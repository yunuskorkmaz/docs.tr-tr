# <a name="the-file-filename-is-not-a-text-file"></a>' ' Dosyası \<filename> bir metin dosyası değil

 **Hata kimliği:** BC2015  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Doğru dosya adını ve yolunu tedarik edin.
