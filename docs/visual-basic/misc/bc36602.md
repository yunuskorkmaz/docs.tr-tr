# <a name="39readonly39-variable-cannot-be-the-target-of-an-assignment-in-a-lambda-expression-inside-a-constructor"></a><span data-ttu-id="492ce-102">&#39; Salt okunur &#39; değişken yapıcı içinde bir lambda ifadesinde atama işleminin hedefi olamaz</span><span class="sxs-lookup"><span data-stu-id="492ce-102">&#39;ReadOnly&#39; variable cannot be the target of an assignment in a lambda expression inside a constructor</span></span>
<span data-ttu-id="492ce-103">Başvurulan bir `ReadOnly` izin lambda ifadesi içinde değişken.</span><span class="sxs-lookup"><span data-stu-id="492ce-103">You have referred to a `ReadOnly` variable from within a lambda expression, which is not permitted.</span></span> <span data-ttu-id="492ce-104">Aşağıdaki kod, göndererek bu hataya neden olur `ReadOnly` değişkeni `m` bağımsız değişkeni olarak bir `ByRef` parametresi.</span><span class="sxs-lookup"><span data-stu-id="492ce-104">The following code causes this error by sending the `ReadOnly` variable `m` as the argument to a `ByRef` parameter.</span></span>  
  
```vb  
Class Class1  
  
    ReadOnly m As Integer  
  
    Sub New()  
        ' The use of m triggers the error.  
        Dim f = Function() Test(m)  
    End Sub  
  
    Function Test(ByRef n As Integer) As String  
    End Function  
  
End Class  
```  
  
 <span data-ttu-id="492ce-105">**Hata Kimliği:** BC36602</span><span class="sxs-lookup"><span data-stu-id="492ce-105">**Error ID:** BC36602</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="492ce-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="492ce-106">To correct this error</span></span>  
  
-   <span data-ttu-id="492ce-107">Mümkünse, parametre değiştirme `n` işlevindeki `Test` için bir `ByVal` aşağıdaki kodda gösterildiği gibi parametre.</span><span class="sxs-lookup"><span data-stu-id="492ce-107">If possible, change parameter `n` in function `Test` to a `ByVal` parameter, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix1  
  
        ReadOnly m As Integer  
  
        Sub New()  
            Dim f = Function() Test(m)  
        End Sub  
  
        Function Test(ByVal n As Integer) As String  
        End Function  
  
    End Class  
    ```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="492ce-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="492ce-108">To correct this error</span></span>  
  
-   <span data-ttu-id="492ce-109">Ata `ReadOnly` değişkeni `m` Oluşturucu, yerel bir değişkene ve yerine yerel değişkenin `m`aşağıdaki kodda gösterildiği gibi.</span><span class="sxs-lookup"><span data-stu-id="492ce-109">Assign the `ReadOnly` variable `m` to a local variable in the constructor, and use the local variable in place of `m`, as shown in the following code.</span></span>  
  
    ```vb  
    Class Class1Fix2  
         ReadOnly m As Integer  
  
        Sub New()  
            Dim temp = m  
            Dim f = Function() Test(temp)  
        End Sub  
  
        Function Test(ByRef n As Integer) As String  
        End Function  
  
    End Class  
    ```
