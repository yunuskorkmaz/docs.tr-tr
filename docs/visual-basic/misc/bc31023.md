# <a name="unable-to-parse-xml-lterrorgt"></a>XML ayrıştırılamıyor: &lt;hata&gt;
Kullanılan XML dosyasında hata var.  
  
 **Hata Kimliği:** BC31023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML düzgün biçimlendirme, geçerli olduğundan ve dosyanın bozuk olmadığını doğrulayın.
