# <a name="expression-expected"></a>İfade bekleniyordu
Gibi bir işleç `+` veya `=` kendi sağ işleneni, bir ifade gerektirir, ancak hiçbir tanınabilir ifade görüntülenir.  
  
 **Hata Kimliği:** BC30201  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İşlecinin sağ işleneni için geçerli bir ifade girin.
