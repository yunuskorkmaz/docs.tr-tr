# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39; Let &#39; ve &#39; Set &#39; atama deyimleri artık desteklenmiyor
`Let`ve `Set` atamalarını kullanılamaz.  
  
 **Hata Kimliği:** BC30807  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Atlayın `Let` ve `Set` atama deyiminden; örneğin:  
  
     `X = 2`
