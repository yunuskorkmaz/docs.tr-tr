# <a name="expected-closing-39gt39-for-xml-cdata-section"></a>Kapatma beklenen &#39;]]&gt; &#39; XML CDATA bölümünün
XML CDATA değişmez değeri gerekli kapanış eksik ']] >' karakter.  
  
 **Hata Kimliği:** BC31162  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli Kapanış Ekle ']] >' XML CDATA değişmez değeri sonuna karakter.
