<span data-ttu-id="26d61-103">Bir bildirim deyimi, oluşturulacak tür olarak bir tür parametresi belirten [Yeni bir işleç](../language-reference/operators/new-operator.md) yan tümcesi kullanır ve tür parametresi kısıtlama olmadan bildirilmiştir `New` .</span><span class="sxs-lookup"><span data-stu-id="26d61-103">A declaration statement uses a [New Operator](../language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="26d61-104">Tür parametresindeki bir *kısıtlama* , genel tür oluşturulduğunda bu tür parametresine geçirilen tür bağımsız değişkenine bir gereksinim uygular.</span><span class="sxs-lookup"><span data-stu-id="26d61-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="26d61-105">`New`Kısıtlama, tür bağımsız değişkeninin, oluşturma kodunun erişebileceği parametresiz bir oluşturucuyu kullanıma sunması gerektiğini belirtir.</span><span class="sxs-lookup"><span data-stu-id="26d61-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="26d61-106">Bu, `New` bildirim deyimindeki bir yan tümcesinin bu türün bir örneğini oluşturmasına izin verir.</span><span class="sxs-lookup"><span data-stu-id="26d61-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="26d61-107">**Hata kimliği:** BC32046</span><span class="sxs-lookup"><span data-stu-id="26d61-107">**Error ID:** BC32046</span></span>
