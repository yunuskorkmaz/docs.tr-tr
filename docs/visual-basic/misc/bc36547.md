<span data-ttu-id="7ff56-103">Bir özellik adı yalnızca bir kez anonim bir tür bildiriminde kurulabilir.</span><span class="sxs-lookup"><span data-stu-id="7ff56-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="7ff56-104">Örneğin, aşağıdaki bildirimi geçerli değil:</span><span class="sxs-lookup"><span data-stu-id="7ff56-104">For example, the following declarations are not valid:</span></span>

'' Name.
