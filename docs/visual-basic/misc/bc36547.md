# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="8b984-102">Anonim tür üyesi veya ' \<propertyname> ' özelliği zaten tanımlanmış</span><span class="sxs-lookup"><span data-stu-id="8b984-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="8b984-103">Bir özellik adı, anonim bir türün bildiriminde yalnızca bir kez oluşturulabilir.</span><span class="sxs-lookup"><span data-stu-id="8b984-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="8b984-104">Örneğin, aşağıdaki bildirimler geçerli değildir:</span><span class="sxs-lookup"><span data-stu-id="8b984-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="8b984-105">**Hata kimliği:** BC36547</span><span class="sxs-lookup"><span data-stu-id="8b984-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8b984-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="8b984-106">To correct this error</span></span>  
  
- <span data-ttu-id="8b984-107">Özelliklerden biri için farklı bir ad seçin.</span><span class="sxs-lookup"><span data-stu-id="8b984-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="8b984-108">Ad ve değerleri verdiğiniz değişkenler veya özellik adları için yeni adlar sağlayın.</span><span class="sxs-lookup"><span data-stu-id="8b984-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
