## <a name="changes-during-reference-type-conversions"></a><span data-ttu-id="7bfea-179">Başvuru türü dönüştürmeleri sırasında yapılan değişiklikler</span><span class="sxs-lookup"><span data-stu-id="7bfea-179">Changes During Reference Type Conversions</span></span>
