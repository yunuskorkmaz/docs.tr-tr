# <a name="nullable-types-c-programming-guide"></a><span data-ttu-id="e90c1-103">Boş değer atanabilir türler (C# programlama Kılavuzu)</span><span class="sxs-lookup"><span data-stu-id="e90c1-103">Nullable types (C# Programming Guide)</span></span>

<span data-ttu-id="e90c1-104">Boş değer atanabilir türler örnekleridir <xref:System.Nullable%601?displayProperty=nameWithType> yapısı.</span><span class="sxs-lookup"><span data-stu-id="e90c1-104">Nullable types are instances of the <xref:System.Nullable%601?displayProperty=nameWithType> struct.</span></span> <span data-ttu-id="e90c1-105">Boş değer atanabilir türler bir temel türünün tüm değerleri temsil eden `T`ve ek [null](../../language-reference/keywords/null.md) değeri.</span><span class="sxs-lookup"><span data-stu-id="e90c1-105">Nullable types can represent all the values of an underlying type `T`, and an additional [null](../../language-reference/keywords/null.md) value.</span></span> <span data-ttu-id="e90c1-106">Temel alınan türü `T` alamayan herhangi olabilir [değer türü](../../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="e90c1-106">The underlying type `T` can be any non-nullable [value type](../../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="e90c1-107">`T` bir başvuru türü olamaz.</span><span class="sxs-lookup"><span data-stu-id="e90c1-107">`T` cannot be a reference type.</span></span>

<span data-ttu-id="e90c1-108">Örneğin, atayabilirsiniz `null` veya tamsayı değerdeki <xref:System.Int32.MinValue?displayProperty=nameWithType> için <xref:System.Int32.MaxValue?displayProperty=nameWithType> için bir `Nullable<int>` ve [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md), veya `null` için bir `Nullable<bool>`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-108">For example, you can assign `null` or any integer value from <xref:System.Int32.MinValue?displayProperty=nameWithType> to <xref:System.Int32.MaxValue?displayProperty=nameWithType> to a `Nullable<int>` and [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md), or `null` to a `Nullable<bool>`.</span></span>

<span data-ttu-id="e90c1-109">Boş değer atanabilir bir tür, bir temel türü tanımlanmamış değerini temsil etmek, ihtiyacınız olduğunda kullanın.</span><span class="sxs-lookup"><span data-stu-id="e90c1-109">You use a nullable type when you need to represent the undefined value of an underlying type.</span></span> <span data-ttu-id="e90c1-110">Bir Boolean değişkeni yalnızca iki değere sahip olabilir: true ve false.</span><span class="sxs-lookup"><span data-stu-id="e90c1-110">A Boolean variable can have only two values: true and false.</span></span> <span data-ttu-id="e90c1-111">"Tanımlanmamış" değer yoktur.</span><span class="sxs-lookup"><span data-stu-id="e90c1-111">There is no "undefined" value.</span></span> <span data-ttu-id="e90c1-112">Birçok programlama uygulamada, bir değişken değeri en önemlisi veritabanı etkileşimleri tanımsız veya eksik olabilir.</span><span class="sxs-lookup"><span data-stu-id="e90c1-112">In many programming applications, most notably database interactions, a variable value can be undefined or missing.</span></span> <span data-ttu-id="e90c1-113">Örneğin, bir veritabanındaki bir alanı true veya false değerleri içerebilir veya hiç değer içerebilir.</span><span class="sxs-lookup"><span data-stu-id="e90c1-113">For example, a field in a database may contain the values true or false, or it may contain no value at all.</span></span> <span data-ttu-id="e90c1-114">Kullandığınız bir `Nullable<bool>` durumda yazın.</span><span class="sxs-lookup"><span data-stu-id="e90c1-114">You use a `Nullable<bool>` type in that case.</span></span>

<span data-ttu-id="e90c1-115">Boş değer atanabilir türler aşağıdaki özelliklere sahiptir:</span><span class="sxs-lookup"><span data-stu-id="e90c1-115">Nullable types have the following characteristics:</span></span>
  
- <span data-ttu-id="e90c1-116">Boş değer atanabilir türleri temsil eden atanabilir değer türü değişkenler `null` değeri.</span><span class="sxs-lookup"><span data-stu-id="e90c1-116">Nullable types represent value-type variables that can be assigned the `null` value.</span></span> <span data-ttu-id="e90c1-117">Bir başvuru türüne göre boş değer atanabilir bir tür oluşturulamıyor.</span><span class="sxs-lookup"><span data-stu-id="e90c1-117">You cannot create a nullable type based on a reference type.</span></span> <span data-ttu-id="e90c1-118">(Başvuru türleri halihazırda desteklediği `null` değeri.)</span><span class="sxs-lookup"><span data-stu-id="e90c1-118">(Reference types already support the `null` value.)</span></span>  
  
- <span data-ttu-id="e90c1-119">Söz dizimi `T?` için toplu özelliktir `Nullable<T>`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-119">The syntax `T?` is shorthand for `Nullable<T>`.</span></span> <span data-ttu-id="e90c1-120">İki tür birbirinin yerine kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="e90c1-120">The two forms are interchangeable.</span></span>  
  
- <span data-ttu-id="e90c1-121">Temel alınan bir değer türü için yaptığınız gibi boş değer atanabilir bir tür için bir değer atayın: `int? x = 10;` veya `double? d = 4.108;`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-121">Assign a value to a nullable type just as you would for an underlying value type: `int? x = 10;` or `double? d = 4.108;`.</span></span> <span data-ttu-id="e90c1-122">Ayrıca atayabilirsiniz `null` değer: `int? x = null;`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-122">You also can assign the `null` value: `int? x = null;`.</span></span>  
  
- <span data-ttu-id="e90c1-123">Kullanım <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> ve <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> salt okunur özellikler için null test etmek ve değerini, aşağıdaki örnekte gösterildiği gibi almak için: `if (x.HasValue) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="e90c1-123">Use the <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> and <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> readonly properties to test for null and retrieve the value, as shown in the following example: `if (x.HasValue) y = x.Value;`</span></span>  
  
  - <span data-ttu-id="e90c1-124"><xref:System.Nullable%601.HasValue%2A> Özelliği döndürür `true` değişken bir değer içeriyorsa veya `false` ise `null`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-124">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>
  
  - <span data-ttu-id="e90c1-125"><xref:System.Nullable%601.Value%2A> Özelliği, bir değer döndürür <xref:System.Nullable%601.HasValue%2A> döndürür `true`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-125">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="e90c1-126">Aksi takdirde, bir <xref:System.InvalidOperationException> oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="e90c1-126">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>  
  
- <span data-ttu-id="e90c1-127">Ayrıca `==` ve `!=` işleçleri aşağıdaki örnekte gösterildiği gibi boş değer atanabilir bir tür ile: `if (x != null) y = x.Value;`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-127">You can also use the `==` and `!=` operators with a nullable type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="e90c1-128">Varsa `a` ve `b` her ikisi de null olan `a == b` değerlendiren `true`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-128">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>  

- <span data-ttu-id="e90c1-129">C# 7.0 ile başlayarak, desen eşleştirme hem incelemek ve boş değer atanabilir bir tür değerini almak için kullanabilirsiniz: `if (x is int xValue) y = xValue;`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-129">Beginning with C# 7.0, you can use pattern matching to both examine and get a value of a nullable type: `if (x is int xValue) y = xValue;`.</span></span>
  
- <span data-ttu-id="e90c1-130">Varsayılan değer olan `T?` bir örneği olan <xref:System.Nullable%601.HasValue%2A> özelliği döndürür `false`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-130">The default value of `T?` is an instance whose <xref:System.Nullable%601.HasValue%2A> property returns `false`.</span></span>  

- <span data-ttu-id="e90c1-131">Kullanım <xref:System.Nullable%601.GetValueOrDefault> ya da atanan değer döndürmek için yöntemi veya [varsayılan](../../language-reference/keywords/default-values-table.md) değer boş değer atanabilir türde ise temel değer türünün `null`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-131">Use the <xref:System.Nullable%601.GetValueOrDefault> method to return either the assigned value, or the [default](../../language-reference/keywords/default-values-table.md) value of the underlying value type if the value of the nullable type is `null`.</span></span>  

- <span data-ttu-id="e90c1-132">Kullanım <xref:System.Nullable%601.GetValueOrDefault(%600)> atanan değer veya sağlanan varsayılan değer boş değer atanabilir tür değeri ise döndürülecek yöntemi `null`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-132">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>
  
- <span data-ttu-id="e90c1-133">Kullanım [null birleşim işleci](../../language-reference/operators/null-coalescing-operator.md), `??`, boş değer atanabilir türde bir değere göre bir temel türü bir değer atamak için: `int? x = null; int y = x ?? -1;`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-133">Use the [null-coalescing operator](../../language-reference/operators/null-coalescing-operator.md), `??`, to assign a value to an underlying type based on a value of the nullable type: `int? x = null; int y = x ?? -1;`.</span></span> <span data-ttu-id="e90c1-134">Örnekte, bu yana `x` null, sonuç değeri olan `y` olduğu `-1`.</span><span class="sxs-lookup"><span data-stu-id="e90c1-134">In the example, since `x` is null, the result value of `y` is `-1`.</span></span>

- <span data-ttu-id="e90c1-135">Bir kullanıcı tanımlı dönüştürme iki veri türleri arasında tanımlıysa, bu veri türlerini boş değer atanabilir sürümleriyle aynı dönüştürme de kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="e90c1-135">If a user-defined conversion is defined between two data types, the same conversion can also be used with the nullable versions of these data types.</span></span>
  
- <span data-ttu-id="e90c1-136">İç içe geçmiş boş değer atanabilir türler izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="e90c1-136">Nested nullable types are not allowed.</span></span> <span data-ttu-id="e90c1-137">Aşağıdaki satırı derleme değil: `Nullable<Nullable<int>> n;`</span><span class="sxs-lookup"><span data-stu-id="e90c1-137">The following line doesn't compile: `Nullable<Nullable<int>> n;`</span></span>
