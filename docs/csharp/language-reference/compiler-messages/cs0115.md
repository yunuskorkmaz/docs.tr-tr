## <a name="example"></a><span data-ttu-id="3fccc-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="3fccc-106">Example</span></span>  
 <span data-ttu-id="3fccc-107">Aşağıdaki örnek CS0115 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3fccc-107">The following sample generates CS0115.</span></span> <span data-ttu-id="3fccc-108">CS0115 iki şekilde bir çözüm oluşturabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="3fccc-108">You can resolve CS0115 in one of two ways:</span></span>  
  
- <span data-ttu-id="3fccc-109">Anahtar sözcüğünü içindeki `MyClass2`yönteminden kaldırın. `override`</span><span class="sxs-lookup"><span data-stu-id="3fccc-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
- <span data-ttu-id="3fccc-110">`MyClass1` İçin`MyClass2`temel sınıf olarak kullanın.</span><span class="sxs-lookup"><span data-stu-id="3fccc-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
