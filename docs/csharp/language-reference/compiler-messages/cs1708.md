# <a name="compiler-error-cs1708"></a>Derleyici Hatası CS1708
Sabit boyut arabelleklerine yalnızca yerel halk veya alanlar üzerinden erişilebilir  
  
 C# 2.0'daki yeni bir özellik, bir `struct`.'ün içinde bir satır içi diziyi tanımlayabilme yeteneğidir. Bu tür dizilere yalnızca yerel değişkenler veya alanlar üzerinden erişilebilir ve bir ifadenin sol tarafındaki ara değerler olarak başvurulabilir. Ayrıca, diziler veya `static` . `readonly`  
  
 Bu hatayı gidermek için bir dizi değişkeni tanımlayın ve satır içi diziyi değişkene atayın. Veya, satır `static` `readonly` içi diziyi temsil eden alandan veya değiştiriyi kaldırın.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1708 oluşturur.  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
