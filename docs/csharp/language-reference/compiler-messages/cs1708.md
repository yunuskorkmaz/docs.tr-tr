# <a name="compiler-error-cs1708"></a><span data-ttu-id="8a747-103">Derleyici Hatası CS1708</span><span class="sxs-lookup"><span data-stu-id="8a747-103">Compiler Error CS1708</span></span>
<span data-ttu-id="8a747-104">Sabit boyutlu arabelleklere yalnızca yerel öğeler veya alanlar üzerinden erişilebilir</span><span class="sxs-lookup"><span data-stu-id="8a747-104">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="8a747-105">C# 2,0 ' deki yeni bir özellik, bir satır içi diziyi bir içinde tanımlayabilme yeteneğidir `struct` .</span><span class="sxs-lookup"><span data-stu-id="8a747-105">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="8a747-106">Bu tür dizilere yalnızca yerel değişkenler veya alanlar aracılığıyla erişilebilir ve bir ifadenin sol tarafında ara değer olarak başvurulmayabilir.</span><span class="sxs-lookup"><span data-stu-id="8a747-106">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="8a747-107">Ayrıca, dizilere veya olan alanlar tarafından erişilemez `static` `readonly` .</span><span class="sxs-lookup"><span data-stu-id="8a747-107">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="8a747-108">Bu hatayı çözmek için bir dizi değişkeni tanımlayın ve satır içi diziyi değişkenine atayın.</span><span class="sxs-lookup"><span data-stu-id="8a747-108">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="8a747-109">Ya da, `static` `readonly` satır içi diziyi temsil eden alandan veya değiştiricisini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="8a747-109">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8a747-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="8a747-110">Example</span></span>  
 <span data-ttu-id="8a747-111">Aşağıdaki örnek CS1708 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="8a747-111">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
