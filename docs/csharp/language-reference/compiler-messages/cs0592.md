# <a name="compiler-error-cs0592"></a>Derleyici Hatası CS0592

' Attribute ' özniteliği bu bildirim türünde geçerli değil. Yalnızca ' tür ' bildirimlerinde geçerlidir.  
  
 Bir öznitelik tanımladığınızda, bir değer belirterek hangi yapıların uygulanabileceğini tanımlarsınız `AttributeTargets` . Aşağıdaki örnekte, özniteliği, `MyAttribute` özniteliği belirttiğinden yalnızca arayüzlere uygulanabilir `AttributeUsage` `AttributeTargets.Interface` . Özniteliği bir sınıfa (sınıf) uygulandığından hata oluşturulur `A` .  
  
## <a name="example"></a>Örnek

 Aşağıdaki örnek CS0592 oluşturur:  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.
public class A
{  
    public static void Main()  
    {  
    }  
}  
```
