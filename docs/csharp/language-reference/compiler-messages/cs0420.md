# <a name="compiler-warning-level-1-cs0420"></a>Derleyici Uyarısı (düzey 1) CS0420

' tanımlayıcı ': geçici alana yapılan başvuru geçici olarak değerlendirilmeyecektir  
  
 Geçici bir alan, `ref` işlevin kapsamı içinde geçici olarak değerlendirilmeyeceği için normalde bir veya **Out** parametresi kullanılarak geçirilmemelidir. Bunun için, birbirine kenetlenmiş bir API çağrılırken olduğu gibi özel durumlar vardır. Herhangi bir uyarıda olduğu gibi, kasıtlı olarak geçici bir alanı bir başvuru parametresi olarak kullandığınız nadir durumlarda bu uyarıyı devre dışı bırakmak için [#pragma uyarısını](../preprocessor-directives/preprocessor-pragma-warning.md) kullanabilirsiniz.  
  
 Aşağıdaki örnek CS0420 oluşturur:  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
