# <a name="compiler-error-cs0433"></a><span data-ttu-id="ada6a-102">Derleyici Hatası CS0433</span><span class="sxs-lookup"><span data-stu-id="ada6a-102">Compiler Error CS0433</span></span>

 <span data-ttu-id="ada6a-104">Uygulamanızda başvurulan iki farklı derleme aynı ad alanına ve belirsizlik üreten türü içerir.</span><span class="sxs-lookup"><span data-stu-id="ada6a-104">Two different assemblies referenced in your application contain the same namespace and type, which produces ambiguity.</span></span>

## <a name="example"></a><span data-ttu-id="ada6a-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="ada6a-110">Example</span></span>  
 <span data-ttu-id="ada6a-111">Aşağıdaki örnek CS0433 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="ada6a-111">The following example generates CS0433.</span></span>  
  
```csharp

using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ada6a-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="ada6a-112">Example</span></span>
