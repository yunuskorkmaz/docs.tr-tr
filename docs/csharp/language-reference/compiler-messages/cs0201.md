```csharp
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.

   }  
}  
```

## <a name="example"></a><span data-ttu-id="5bb59-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="5bb59-110">Example</span></span>

 <span data-ttu-id="5bb59-111">Aşağıdaki örnek, bir artırma işlemi tarafından parametreli olsa da, tek başına işaretlenen bir ifade olmadığından, CS0201 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="5bb59-111">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>  

```csharp
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;
         }  
      }  
   }  
}  
```
