{  
   public static void Main()

## <a name="example"></a><span data-ttu-id="111d1-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="111d1-110">Example</span></span>

 <span data-ttu-id="111d1-111">Aşağıdaki örnek CS0201 oluşturur çünkü tarafından denetlenen bir Artım işlemi tarafından parametre haline rağmen kendisini bir deyimi değil.</span><span class="sxs-lookup"><span data-stu-id="111d1-111">The following sample generates CS0201 because checked by itself is not a statement, even though it is parameterized by an increment operation.</span></span>

   public void Add(T x)

      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK

            i++;
         }  
      }  
   }
