# <a name="compiler-error-cs0122"></a>Derleyici Hatası CS0122

'üye' koruma seviyesi nedeniyle erişilemez

 Bir sınıf üyesinin [erişim değiştiricisi](/dotnet/csharp/language-reference/keywords) üyeye erişimi engeller. Daha fazla bilgi için [Erişim Değiştiriciler'e](../../programming-guide/classes-and-structs/access-modifiers.md)bakın.

 Bunun bir nedeni (aşağıdaki örnekte gösterilmez) bir arkadaş derlemesi hedefinde **/out** derleyici bayrağı atlayarak olabilir. Daha fazla bilgi için [Arkadaş Derlemeleri](../../../standard/assembly/friend.md) ve [-out (C# Derleyici Seçenekleri)](../compiler-options/out-compiler-option.md)bakın.

## <a name="example"></a>Örnek

 Aşağıdaki örnek CS0122 oluşturur:

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
