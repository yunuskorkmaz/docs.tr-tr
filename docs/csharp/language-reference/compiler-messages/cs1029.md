# <a name="compiler-error-cs1029"></a>Derleyici Hatası CS1029

\#Hata: ' metin '

[#Error](../preprocessor-directives/preprocessor-error.md) yönergesiyle tanımlanan bir hatanın metnini görüntüler.

Aşağıdaki örnek, Kullanıcı tanımlı bir hatanın nasıl oluşturulacağını gösterir:

```csharp
// CS1029.cs
class Sample
{
   static void Main()
   {
      #error Let's give an error here   // CS1029
   }
}
```

Derleme aşağıdaki çıktıyı üretir:

```console
example.cs(9,8): error CS1029: #error: 'Let's give an error here   // CS1029  '
```
