# <a name="compiler-error-cs1519"></a><span data-ttu-id="a96c0-103">Derleyici Hatası CS1519</span><span class="sxs-lookup"><span data-stu-id="a96c0-103">Compiler Error CS1519</span></span>

<span data-ttu-id="a96c0-104">Sınıf, yapı veya arabirim üye bildiriminde geçersiz belirteç ' token '</span><span class="sxs-lookup"><span data-stu-id="a96c0-104">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="a96c0-105">Bu hata, ait olmadığı bir konumda bir belirtece her karşılaşıldığında üretilir.</span><span class="sxs-lookup"><span data-stu-id="a96c0-105">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="a96c0-106">*Belirteç* bir anahtar sözcüktür; tanımlayıcı (bir sınıf, yapı, yöntem, vb.); 108, "Merhaba" veya ' A ' gibi bir dize, karakter veya sayısal değişmez değer. ya da veya gibi bir işleç veya noktalama `==` işareti `;` .</span><span class="sxs-lookup"><span data-stu-id="a96c0-106">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="a96c0-107">Tür için geçersiz değiştiriciler içeren herhangi bir [sınıf](../keywords/class.md), yapı veya arabirim üye bildirimi, bu hatayı oluşturacaktır.</span><span class="sxs-lookup"><span data-stu-id="a96c0-107">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="a96c0-108">Hatayı onarmak için geçersiz değiştiricilerini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="a96c0-108">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="a96c0-109">Aşağıdaki örnek beş yerde CS1519 oluşturur, çünkü belirteçler geçerli olmadıkları konumlara yerleştirilir:</span><span class="sxs-lookup"><span data-stu-id="a96c0-109">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
