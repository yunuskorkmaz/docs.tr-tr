# <a name="compiler-warning-level-2-cs1710"></a><span data-ttu-id="9fd18-102">Derleyici Uyarısı (Düzey 2) CS1710</span><span class="sxs-lookup"><span data-stu-id="9fd18-102">Compiler Warning (level 2) CS1710</span></span>

 <span data-ttu-id="9fd18-104">Genel bir tür belgeleri tür parametresi için bir yinelenen etiket içerir.</span><span class="sxs-lookup"><span data-stu-id="9fd18-104">The documentation of a generic type includes a duplicate tag for the type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9fd18-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="9fd18-105">Example</span></span>  
 <span data-ttu-id="9fd18-106">Aşağıdaki kod görünmesi CS1710 uyarı neden olur.</span><span class="sxs-lookup"><span data-stu-id="9fd18-106">The following code will cause warning CS1710 to appear.</span></span>  
  
```csharp  
// CS1710.cs  
// compile with: /doc:cs1710.xml  
// To resolve this warning, delete one of the duplicate <typeparam>'s.  
using System;  
class Stack<ItemType>  
{  
}  
  
/// <typeparam name="MyType">can be an int</typeparam>  
/// <typeparam name="MyType">can be an int</typeparam>  
class MyStackWrapper<MyType>  
{  
    // Open constructed type Stack<MyType>.  
    Stack<MyType> stack;  
    public MyStackWrapper(Stack<MyType> s)  
    {  
        stack = s;  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        // Closed constructed type Stack<int>.  
        Stack<int> stackInt = new Stack<int>();  
        MyStackWrapper<int> MyStackWrapperInt =  
            new MyStackWrapper<int>(stackInt);  
    }  
}  
```
