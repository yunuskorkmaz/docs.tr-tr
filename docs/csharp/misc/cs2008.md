# <a name="compiler-error-cs2008"></a>Derleyici hatası CS2008

Giriş belirtilmedi  
  
 Derleyici çağrıldı ve derleyici seçenekleri belirtildi, ancak hiçbir kaynak kodu dosyası geçirilmedi.
