# <a name="compiler-error-cs0198"></a><span data-ttu-id="326de-102">Derleyici Hatası CS0198</span><span class="sxs-lookup"><span data-stu-id="326de-102">Compiler Error CS0198</span></span>
<span data-ttu-id="326de-103">Statik salt okunur alan 'name' alanlarının (statik oluşturucu veya değişken başlatıcı içinde olması dışında) atanamaz</span><span class="sxs-lookup"><span data-stu-id="326de-103">Fields of static readonly field 'name' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="326de-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) değişken aynı olması gerekir [statik](../../csharp/language-reference/keywords/static.md) kullanım başlatmak istediğiniz oluşturucusu olarak.</span><span class="sxs-lookup"><span data-stu-id="326de-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="326de-105">Daha fazla bilgi için bkz: [statik oluşturucular](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="326de-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="326de-106">Aşağıdaki örnek CS0198 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="326de-106">The following sample generates CS0198:</span></span>  
  
```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
