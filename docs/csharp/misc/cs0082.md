## <a name="example"></a><span data-ttu-id="ef910-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="ef910-106">Example</span></span>  
 <span data-ttu-id="ef910-107">Aşağıdaki örnek CS0082 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ef910-107">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
