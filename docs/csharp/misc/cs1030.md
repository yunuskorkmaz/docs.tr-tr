# <a name="compiler-warning-level-1-cs1030"></a>Derleyici Uyarısı (düzey 1) CS1030
\#uyarı: 'metin'  
  
 [#warning](../language-reference/preprocessor-directives/preprocessor-warning.md) yönergesi ile tanımlanan bir uyarı metnini görüntüler.  
  
 Aşağıdaki örnek, kullanıcı tanımlı bir uyarının nasıl oluşturulabildiğini gösterir:  
  
```csharp  
// CS1030.cs  
class Sample  
{  
   static void Main()  
   {  
      #warning Let's give a warning here
   }  
}  
```

Derleme aşağıdaki çıktıyı üretir:

```console
example.cs(6,16): warning CS1030: #warning: 'Let's give a warning here'
```
