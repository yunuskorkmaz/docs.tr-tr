<span data-ttu-id="939f4-105">Bu uyarı, derleyicinin bir değişkene hiçbir şekilde başvurulduğunu doğrulayabileceği çeşitli senaryolarda oluşabilir:</span><span class="sxs-lookup"><span data-stu-id="939f4-105">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>
