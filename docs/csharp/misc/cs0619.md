# <a name="compiler-error-cs0619"></a><span data-ttu-id="23f6e-102">Derleyici Hatası CS0619</span><span class="sxs-lookup"><span data-stu-id="23f6e-102">Compiler Error CS0619</span></span>
<span data-ttu-id="23f6e-103">'member' artık kullanılmıyor: 'text'</span><span class="sxs-lookup"><span data-stu-id="23f6e-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="23f6e-104">Sınıf üyesi [kullanılmaz] işaretlendi ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) öznitelik sağlayacak şekilde sınıf üyesine başvuru olduğunda bir hata verilir.</span><span class="sxs-lookup"><span data-stu-id="23f6e-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="23f6e-105">Aşağıdaki örnek, CS0619 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="23f6e-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
