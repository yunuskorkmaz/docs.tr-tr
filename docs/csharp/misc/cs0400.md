 <span data-ttu-id="99f06-107">Bu hatayı önlemek için tanımlayıcının bildirimini bulun, doğru yazımı doğrulayın ve bildirim ayrı bir derlemedeyse, uygun derleme başvurusuna sahip olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="99f06-107">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="99f06-108">Tanımlayıcı başka bir tür veya ad alanı içinde bildirilirse, aşağıdakilerden sonra tam nitelikli adı kullanın:</span><span class="sxs-lookup"><span data-stu-id="99f06-108">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="99f06-109">Aşağıdaki örnek CS0400 üretir:</span><span class="sxs-lookup"><span data-stu-id="99f06-109">The following sample generates CS0400:</span></span>

   }
