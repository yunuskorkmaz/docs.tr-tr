# <a name="compiler-error-cs0591"></a>Derleyici hatası CS0591
' Attribute ' özniteliğine yönelik bağımsız değişken için geçersiz değer  
  
 Bir özniteliğe geçersiz bir bağımsız değişken veya birbirini dışlayan iki bağımsız değişken geçirildi.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0591 oluşturur:  
  
```csharp  
// CS0591.cs  
using System;  
  
[AttributeUsage(0)]   // CS0591  
class I: Attribute  
{  
}  
  
public class a  
{  
    public static void Main()  
    {  
    }  
}  
```
