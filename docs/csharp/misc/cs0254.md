# <a name="compiler-error-cs0254"></a>Derleyici Hatası CS0254
Fixed deyimi atamasının sağ taraftaki cast ifadesi olmayabilir  
  
 Sağ tarafındaki bir [sabit](../../csharp/language-reference/keywords/fixed-statement.md) ifadesi bir cast kullanamazsınız. Daha fazla bilgi için bkz: [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Aşağıdaki örnek CS0254 oluşturur:  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
