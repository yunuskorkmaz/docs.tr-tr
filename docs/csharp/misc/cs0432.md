# <a name="compiler-error-cs0432"></a><span data-ttu-id="cd5aa-102">Derleyici Hatası CS0432</span><span class="sxs-lookup"><span data-stu-id="cd5aa-102">Compiler Error CS0432</span></span>
<span data-ttu-id="cd5aa-103">Diğer ad 'tanımlayıcısı bulunamadı'</span><span class="sxs-lookup"><span data-stu-id="cd5aa-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="cd5aa-104">Bu hata oluşur "::" sağındaki bir diğer ad değil bir tanımlayıcı.</span><span class="sxs-lookup"><span data-stu-id="cd5aa-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="cd5aa-105">Hatayı gidermek için "." yerine.</span><span class="sxs-lookup"><span data-stu-id="cd5aa-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="cd5aa-106">Aşağıdaki örnek CS0432 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cd5aa-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
