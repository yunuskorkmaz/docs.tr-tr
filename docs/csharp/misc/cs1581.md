# <a name="compiler-warning-level-1-cs1581"></a>Derleyici Uyarısı (düzey 1) CS1581
XML açıklaması cref özniteliğinde geçersiz dönüş türü  
  
 Bir yönteme başvurulmaya çalışılırken, derleyici geçersiz bir dönüş türü nedeniyle bir hata algıladı.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1581 oluşturur:  
  
```csharp  
// CS1581.cs  
// compile with: /W:1 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
    /// <summary>help text</summary>  
    public static void Main()  
    {  
    }  
  
    /// <summary>help text</summary>  
    public static explicit operator int(MyClass f)  
    {  
        return 0;  
    }  
}  
  
/// <seealso cref="MyClass.explicit operator intt(MyClass)"/>  // CS1581  
// try the following line instead  
// /// <seealso cref="MyClass.explicit operator int(MyClass)"/>  
public class MyClass2  
{  
}  
```
