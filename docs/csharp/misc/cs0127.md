# <a name="compiler-error-cs0127"></a><span data-ttu-id="939e7-102">Derleyici Hatası CS0127</span><span class="sxs-lookup"><span data-stu-id="939e7-102">Compiler Error CS0127</span></span>
<span data-ttu-id="939e7-103">'Function' void döndürdüğünden, return anahtar sözcüğünü bir object ifadesi izlememelidir</span><span class="sxs-lookup"><span data-stu-id="939e7-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="939e7-104">Bir yöntemle bir [void](../../csharp/language-reference/keywords/void.md) dönüş türü, değer döndüremez.</span><span class="sxs-lookup"><span data-stu-id="939e7-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="939e7-105">Daha fazla bilgi için [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="939e7-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="939e7-106">Aşağıdaki örnek, CS0127 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="939e7-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
