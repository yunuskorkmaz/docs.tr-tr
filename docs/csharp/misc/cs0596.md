# <a name="compiler-error-cs0596"></a>Derleyici hatası CS0596

Guid özniteliği ComImport özniteliğiyle birlikte belirtilmelidir  
  
 **ComImport** özniteliği kullanılırken, **Guid** özniteliği bulunmalıdır.  
  
 Aşağıdaki örnek CS0596 oluşturur:  
  
```csharp  
// CS0596.cs  
using System.Runtime.InteropServices;  
  
namespace x  
{  
   [ComImport]   // CS0596  
   // try the following line to resolve this CS0596  
   // [ComImport, Guid("00000000-0000-0000-0000-000000000001")]  
   public class a  
   {  
   }  
  
   public class b  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
