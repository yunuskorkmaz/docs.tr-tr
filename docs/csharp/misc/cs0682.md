# <a name="compiler-error-cs0682"></a>Derleyici hatası CS0682
dil tarafından desteklenmediği için ' type1 ', ' type2 ' uygulayamaz  
  
 Bu hata, başka bir dilde yazılmış bir arabirimi uygulamaya çalıştığınızda ve derleyici arabirimi desteklemediği zaman oluşur.
