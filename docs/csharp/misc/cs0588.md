# <a name="compiler-error-cs0588"></a>Derleyici Hatası CS0588
Derleyici iç hatası: 'LEX' aşama  
  
 Derleyici beklenmeyen sözdizimi ayrıştırmak için kendi sorunları nedeniyle başarısız olup olmadığını saptamak deneyin. Bu durumda değilse, deneyin [konuşun bize](/visualstudio/ide/talk-to-us).
