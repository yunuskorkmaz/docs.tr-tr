# <a name="compiler-error-cs1593"></a>Derleyici hatası CS1593
' Del ' temsilcisi ' number ' bağımsız değişkeni almaz  
  
 Bir [temsilci](../language-reference/builtin-types/reference-types.md) çağrısına geçirilen bağımsız değişkenlerin sayısı, temsilci bildirimindeki parametre sayısıyla kabul etmez.  
  
 Aşağıdaki örnek CS1593 oluşturur:  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
