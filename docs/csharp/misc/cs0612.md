# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="307db-102">Derleyici uyarısı (düzey 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="307db-102">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="307db-103">'üye' modası geçmiş</span><span class="sxs-lookup"><span data-stu-id="307db-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="307db-104">Sınıf tasarımcısı [Eski özniteliği](../language-reference/attributes/general.md#obsolete-attribute)ile bir üye işaretledi.</span><span class="sxs-lookup"><span data-stu-id="307db-104">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="307db-105">Bu, üyenin sınıfın gelecekteki bir sürümünde desteklenemeyebilir anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="307db-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="307db-106">Aşağıdaki örnek, eski bir üyeye erişimin Nasıl CS0612 oluşturduğunu gösterir:</span><span class="sxs-lookup"><span data-stu-id="307db-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
