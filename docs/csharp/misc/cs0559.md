# <a name="compiler-error-cs0559"></a>Derleyici hatası CS0559
+ + Veya--işlecinin parametre türü kapsayan tür olmalıdır  
  
 Operatör aşırı yüklemesiyle ilgili yöntem bildirimi bazı yönergeleri izlemelidir. + + Ve--işleçleri için, parametresinin, işlecin aşırı yüklendiği türle aynı türde olması gerekir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0559 oluşturur:  
  
```csharp  
// CS0559.cs  
// compile with: /target:library  
public class iii  
{  
   public static implicit operator int(iii x)  
   {  
      return 0;  
   }  
  
   public static implicit operator iii(int x)  
   {  
      return null;  
   }  
  
   public static int operator ++(int aa)   // CS0559  
   // try the following line instead  
   // public static iii operator ++(iii aa)  
   {  
      return (iii)0;  
   }  
}  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0559 oluşturur.  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559  
   public static S operator --(S? s) { return new S(); }   // CS0559  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
