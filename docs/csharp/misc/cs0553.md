# <a name="compiler-error-cs0553"></a><span data-ttu-id="34053-102">Derleyici Hatası CS0553</span><span class="sxs-lookup"><span data-stu-id="34053-102">Compiler Error CS0553</span></span>
<span data-ttu-id="34053-103">'dönüştürme yordamına': kullanıcı tanımlı dönüşüm/taban sınıfı</span><span class="sxs-lookup"><span data-stu-id="34053-103">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="34053-104">Bir taban sınıfın kullanıcı tanımlı dönüşümler değerlere izin verilmiyor; Bu tür bir işleç gerekmez.</span><span class="sxs-lookup"><span data-stu-id="34053-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="34053-105">Aşağıdaki örnek CS0553 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="34053-105">The following sample generates CS0553:</span></span>  
  
```  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
