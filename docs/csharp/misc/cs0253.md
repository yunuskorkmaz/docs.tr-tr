# <a name="compiler-warning-level-2-cs0253"></a>Derleyici Uyarısı (düzey 2) CS0253
Olası istenmeyen başvuru karşılaştırması; bir değer karşılaştırması elde etmek için, 'yazın' yazın için sağ tarafı döküm  
  
 Derleyici bir başvuru karşılaştırması yapıyor. Dizelerin değerini karşılaştırmak istiyorsanız, ifadenin sağ tarafını `type`.  
  
 Aşağıdaki örnek CS0253 üretir:  
  
```csharp
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
