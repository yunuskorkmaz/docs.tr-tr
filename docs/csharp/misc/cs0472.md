<span data-ttu-id="929a7-103">'value2' türündeki bir değer hiçbir zaman 'değer3' türünün 'null'una eşit olmadığından ifadenin sonucu her zaman 'değer1' dir.</span><span class="sxs-lookup"><span data-stu-id="929a7-103">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>

## <a name="example"></a><span data-ttu-id="929a7-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="929a7-105">Example</span></span>  
 <span data-ttu-id="929a7-106">Aşağıdaki örnek CS0472 üretir.</span><span class="sxs-lookup"><span data-stu-id="929a7-106">The following sample generates CS0472.</span></span>  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;
        return counter;  
    }  
}  
```
