<span data-ttu-id="8ff1d-103">Sabit boyutlu arabellek alanı dizi boyutu belirticisinin alan adından sonra olmalıdır</span><span class="sxs-lookup"><span data-stu-id="8ff1d-103">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="8ff1d-104">Normal dizileri, sabit boyutlu arabellekler bildirimi noktada belirtilmesi için sabit bir boyut gerektirir.</span><span class="sxs-lookup"><span data-stu-id="8ff1d-104">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="8ff1d-105">Bu hatayı gidermek için değişmez değer pozitif bir tamsayı veya sabit bir pozitif tamsayı ekleyin ve sonra tanımlayıcısı köşeli ayraç yerleştirin.</span><span class="sxs-lookup"><span data-stu-id="8ff1d-105">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="8ff1d-106">Aşağıdaki örnek CS1641 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8ff1d-106">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
