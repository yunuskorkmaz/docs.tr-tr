 <span data-ttu-id="d5613-104">Bir parametre olarak [çıkışı](../../csharp/language-reference/keywords/out-parameter-modifier.md) anahtar sözcüğü yöntemi gövdesinde bir değer değil atandı.</span><span class="sxs-lookup"><span data-stu-id="d5613-104">A parameter marked with the [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="d5613-105">Daha fazla bilgi için bkz: [parametreleri geçirme](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span><span class="sxs-lookup"><span data-stu-id="d5613-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span></span>

```csharp

   }  
}
```
