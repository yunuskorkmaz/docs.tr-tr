# <a name="compiler-error-cs0172"></a><span data-ttu-id="934d2-102">Derleyici Hatası CS0172</span><span class="sxs-lookup"><span data-stu-id="934d2-102">Compiler Error CS0172</span></span>
<span data-ttu-id="934d2-103">'Type1' ve 'type2' örtük olarak bir başkasına dönüştürme olmadığından, koşullu ifade türü belirlenemiyor</span><span class="sxs-lookup"><span data-stu-id="934d2-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
 <span data-ttu-id="934d2-104">Bir koşullu ifade türleri her iki tarafında dönüştürebilir olmalıdır `:` işleci.</span><span class="sxs-lookup"><span data-stu-id="934d2-104">In a conditional statement, you must be able to convert the types on either side of the `:` operator.</span></span> <span data-ttu-id="934d2-105">Ayrıca, karşılıklı dönüştürme yordamları olamaz; bir dönüştürme yeterlidir.</span><span class="sxs-lookup"><span data-stu-id="934d2-105">Also, there cannot be mutual conversion routines; you only need one conversion.</span></span> <span data-ttu-id="934d2-106">Daha fazla bilgi için bkz: [dönüştürme işleçleri](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="934d2-106">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="934d2-107">Aşağıdaki örnek CS0172 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="934d2-107">The following sample generates CS0172:</span></span>  
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
