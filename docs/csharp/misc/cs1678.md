# <a name="compiler-error-cs1678"></a>Derleyici Hatası CS1678
Parametre 'sayı' türü 'type1' olarak bildirilir, ancak 'type2' olmalıdır  
  
 Bu hata, anonim bir yöntemdeki parametre türü, yöntemi döküm yaptığınız temsilcinin bildiriminden farklı olduğunda oluşur.  
  
 Aşağıdaki örnek CS1678 oluşturur:  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors
{  
   static void Main()
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
