# <a name="compiler-error-cs0835"></a><span data-ttu-id="64083-102">Derleyici Hatası CS0835</span><span class="sxs-lookup"><span data-stu-id="64083-102">Compiler Error CS0835</span></span>
<span data-ttu-id="64083-103">Lambda, tür bağımsız değişkeni 'türü' bir temsilci türü olmayan bir ifade ağacına dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="64083-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="64083-104">Lambda ifadesi bir ifade ağacına dönüştürülürse, ifade ağacına bağımsız değişkeni için bir temsilci türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="64083-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="64083-105">Ayrıca, lambda ifadesi temsilci türüne dönüştürülebilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="64083-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="64083-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="64083-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="64083-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="64083-108">Example</span></span>  
 <span data-ttu-id="64083-109">Aşağıdaki örnek CS0835 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="64083-109">The following example generates CS0835:</span></span>

```csharp  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
