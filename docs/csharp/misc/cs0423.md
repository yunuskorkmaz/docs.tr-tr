<span data-ttu-id="3fc84-104">' Class ' öğesinin ComImport özniteliği olduğundan, ' Method ' extern veya abstract olmalıdır</span><span class="sxs-lookup"><span data-stu-id="3fc84-104">Since 'class' has the ComImport attribute, 'method' must be extern or abstract</span></span>  
  
 <span data-ttu-id="3fc84-105">ComImport özniteliğini belirtmek, sınıf için uygulamanın bir COM modülünden içeri aktarılacağını belirtir.</span><span class="sxs-lookup"><span data-stu-id="3fc84-105">Specifying the ComImport attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="3fc84-106">Ek yöntemler tanımlanamaz.</span><span class="sxs-lookup"><span data-stu-id="3fc84-106">Additional methods may not be defined.</span></span>  
  
 <span data-ttu-id="3fc84-107">Aşağıdaki örnek CS0423 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3fc84-107">The following sample generates CS0423:</span></span>  
  
```csharp  
// CS0423.cs

[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")

{  
  public static void Main()  // CS0423  
  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
