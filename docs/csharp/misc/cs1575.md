# <a name="compiler-error-cs1575"></a>Derleyici Hatası CS1575
Stackalloc ifadesi [] sonra türü gerektirir.  
  
 İstenen ayırma boyutu ile [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), köşeli parantez içine belirtilmesi gerekir.  
  
 Aşağıdaki örnek CS1575 oluşturur:  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
