# <a name="compiler-error-cs0231"></a><span data-ttu-id="a3f50-102">Derleyici Hatası CS0231</span><span class="sxs-lookup"><span data-stu-id="a3f50-102">Compiler Error CS0231</span></span>
<span data-ttu-id="a3f50-103">Param parametresi, resmi parametre listesindeki son parametre olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="a3f50-103">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="a3f50-104">[Param](../language-reference/keywords/params.md) parametresi değişken bağımsız değişken sayılarını destekler ve diğer tüm parametrelerden sonra olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="a3f50-104">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="a3f50-105">Daha fazla bilgi için [Yöntemler'e](../programming-guide/classes-and-structs/methods.md)bakın.</span><span class="sxs-lookup"><span data-stu-id="a3f50-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="a3f50-106">Aşağıdaki örnek CS0231 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a3f50-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
