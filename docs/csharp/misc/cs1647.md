# <a name="compiler-error-cs1647"></a>Derleyici Hatası CS1647
Çok uzun veya karmaşık 'kodu ' derlemek için ifade  
  
 Kodunuzu işleme derleyicide bir yığın taşması oldu. Bu hatayı gidermek için kodunuzu basitleştirin. Kodunuzu geçerliyse, kişi [ürün desteği](http://msdn.microsoft.com/en-us/77e75b8b-817d-45bf-9c38-458930d873b4).
