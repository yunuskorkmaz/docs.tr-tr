# <a name="compiler-error-cs0037"></a>Derleyici hatası CS0037
Null olamayan bir değer türü olduğundan, null değeri ' Type ' değerine dönüştürülemez  
  
 Derleyici bir değer türüne `null` atayamaz; `null`, yalnızca bir [başvuru türüne](../language-reference/keywords/reference-types.md) veya [null yapılabilir bir değer türüne](../language-reference/builtin-types/nullable-value-types.md)atanabilir. [struct](../language-reference/builtin-types/struct.md) bir değer türüdür.
  
 Aşağıdaki örnek CS0037 oluşturur:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
