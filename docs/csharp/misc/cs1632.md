# <a name="compiler-error-cs1632"></a>Derleyici Hatası CS1632
Kontrol anonim bir yöntem veya lambda ifade vücut bırakamaz  
  
 Bu hata, bir atlama deyimi `goto`**(break**, **continue,** vb.) denetimi anonim bir yöntem bloğundan taşımaya çalışırsa oluşur. Anonim bir yöntem bloğu bir işlev gövdesidir ve yalnızca bir iade deyimi yle veya bloğun sonuna ulaşarak çıkarılabilir.  
  
 Aşağıdaki örnek CS1632 oluşturur:  
  
```csharp  
// CS1632.cs  
// compile with: /target:library  
delegate void MyDelegate();  
class MyClass  
{  
   public void Test()  
   {
      for (int i = 0 ; i < 5 ; i++)  
      {  
         MyDelegate d = delegate {  
            break;   // CS1632  
          };
      }  
   }  
}  
```
