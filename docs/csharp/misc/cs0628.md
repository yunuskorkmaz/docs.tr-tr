# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="df49c-102">Derleyici Uyarısı (düzey 4) CS0628</span><span class="sxs-lookup"><span data-stu-id="df49c-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="df49c-103">'member': yeni korumalı üye bildirilen korumalı sınıfında</span><span class="sxs-lookup"><span data-stu-id="df49c-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="df49c-104">A [korumalı](../../csharp/language-reference/keywords/sealed.md) olamaz sınıfını tanıtmak bir [korumalı](../../csharp/language-reference/keywords/protected.md) üyesi başka bir sınıf devralınmalıdır mümkün olduğundan `sealed` sınıfı ve kullanmak `protected` üyesi.</span><span class="sxs-lookup"><span data-stu-id="df49c-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="df49c-105">Aşağıdaki örnek CS0628 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="df49c-105">The following sample generates CS0628:</span></span>  
  
```csharp  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
