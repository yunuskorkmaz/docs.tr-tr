# <a name="compiler-error-cs1631"></a><span data-ttu-id="52b23-103">Derleyici hatası CS1631</span><span class="sxs-lookup"><span data-stu-id="52b23-103">Compiler Error CS1631</span></span>
<span data-ttu-id="52b23-104">Catch yan tümcesinin gövdesinde değer değerlendirilemiyor</span><span class="sxs-lookup"><span data-stu-id="52b23-104">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="52b23-105">Catch yan tümcesinin gövdesinden yield ifadesine izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="52b23-105">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="52b23-106">Bu hatayı önlemek için, yield ifadesini catch yan tümcesinin gövdesi dışına taşıyın.</span><span class="sxs-lookup"><span data-stu-id="52b23-106">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="52b23-107">Aşağıdaki örnek CS1631 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="52b23-107">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }
  
   public static void Main()
   {  
   }  
}  
```
