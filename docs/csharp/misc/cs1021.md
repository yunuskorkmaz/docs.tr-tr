# <a name="compiler-error-cs1021"></a><span data-ttu-id="d92c1-102">Derleyici Hatası CS1021</span><span class="sxs-lookup"><span data-stu-id="d92c1-102">Compiler Error CS1021</span></span>

<span data-ttu-id="d92c1-103">Tam sayı sabit çok büyük.</span><span class="sxs-lookup"><span data-stu-id="d92c1-103">Integral constant is too large</span></span>  
  
<span data-ttu-id="d92c1-104">Bir tamsayı sabit değeri ile temsil edilen bir değerden büyüktür <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="d92c1-104">A value represented by an integer literal is greater than <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span></span>  
  
<span data-ttu-id="d92c1-105">Aşağıdaki örnek, CS1021 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d92c1-105">The following sample generates CS1021:</span></span>  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

<span data-ttu-id="d92c1-106">Aşağıdaki kod, ayrıca CS1021 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d92c1-106">The following code also generates CS1021:</span></span>

```csharp
using System.Numerics;

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```
