## <a name="example"></a><span data-ttu-id="bbc9b-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="bbc9b-109">Example</span></span>  
 <span data-ttu-id="bbc9b-110">Aşağıdaki kod CS0737 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bbc9b-110">The following code generates CS0737:</span></span>  
  
```csharp  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}

{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
