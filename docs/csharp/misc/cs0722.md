# <a name="compiler-error-cs0722"></a>Derleyici Hatası CS0722
'type': statik türler dönüş türleri olarak kullanılamaz  
  
 Statik bir tür bir dönüş türü olarak statik tür örnekleri oluşturulamaz anlamı yoktur.  
  
 Aşağıdaki örnek, CS0722 oluşturur:  
  
```csharp  
// CS0722.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public SC F()  // CS0722  
   {  
      return null;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
