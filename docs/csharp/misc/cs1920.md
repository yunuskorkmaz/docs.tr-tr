# <a name="compiler-error-cs1920"></a><span data-ttu-id="298d0-102">Derleyici hatası CS1920</span><span class="sxs-lookup"><span data-stu-id="298d0-102">Compiler Error CS1920</span></span>
<span data-ttu-id="298d0-103">Öğe başlatıcısı boş olamaz.</span><span class="sxs-lookup"><span data-stu-id="298d0-103">Element initializer cannot be empty.</span></span>  
  
 <span data-ttu-id="298d0-104">Koleksiyon Başlatıcısı, bir dizi öğe başlatıcısından oluşur.</span><span class="sxs-lookup"><span data-stu-id="298d0-104">A collection initializer consists of a sequence of element initializers.</span></span> <span data-ttu-id="298d0-105">Öğe başlatıcılarının atama ifadesi içermediği müddetçe küme ayraçları içine alınması gerekmez.</span><span class="sxs-lookup"><span data-stu-id="298d0-105">The element initializers do not have to be enclosed in braces unless they contain an assignment expression.</span></span> <span data-ttu-id="298d0-106">Ancak, küme ayraçları yaparsanız boş olamaz.</span><span class="sxs-lookup"><span data-stu-id="298d0-106">However, if you do supply braces, they cannot be empty.</span></span> <span data-ttu-id="298d0-107">Öğe başlatıcısı bir nesne başlatıcısiyse, başlatıcı yeni bir nesne oluşturma ifadesi içerdiği sürece küme ayraçları boş olabilir.</span><span class="sxs-lookup"><span data-stu-id="298d0-107">If the element initializer is an object initializer, the braces may be empty as long as the initializer contains a new object creation expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="298d0-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="298d0-108">To correct this error</span></span>

- <span data-ttu-id="298d0-110">İfadenin bir nesne Başlatıcısı olması amaçlanıyorsa, küme ayracın önüne yeni nesne oluşturma ifadesini ekleyin.</span><span class="sxs-lookup"><span data-stu-id="298d0-110">If the expression is intended to be an object initializer, add the new object creation expression in front of the braces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="298d0-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="298d0-111">Example</span></span>  
 <span data-ttu-id="298d0-112">Aşağıdaki örnek CS1920 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="298d0-112">The following example generates CS1920:</span></span>  
  
```csharp  
  // cs1920.cs  
using System.Collections.Generic;  
public class Test  
{  
    public static int Main()  
    {  
        // Error. Empty initializer   
        // for inner list.  
        List<List<int>> collection =  
            new List<List<int>>() { { } }; // CS1920  
  
        // OK. No initializer for inner list.  
        List<List<int>> collection2 =  
            new List<List<int>>() {  };  
  
        // OK. Inner list is initialized   
        // to one List<int> with zero elements.  
        List<List<int>> collection3 =  
            new List<List<int>>() { new List<int> { } };  
        return 0;  
    }  
}  
```
