# <a name="compiler-error-cs0718"></a>Derleyici hatası CS0718

' Type ': statik türler tür bağımsız değişkeni olarak kullanılamaz  
  
 Statik bir tür örneklenemez, genel bir bağımsız değişken olarak kullanılamaz. Bu hatayı çözmek için, statik türü genel bağımsız değişkenden kaldırın.  
  
## <a name="example"></a>Örnek  

 Aşağıdaki örnek CS0718 oluşturur:  
  
```csharp  
// CS0718.cs  
public static class SC  
{  
    public static void F()  
    {  
    }  
}  
  
public class G<T>  
{  
}  
  
public class CMain  
{  
    public static void Main()  
    {  
        G<SC> gsc = new G<SC>();  // CS0718  
    }  
}  
```
