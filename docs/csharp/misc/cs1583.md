# <a name="compiler-error-cs1583"></a>Derleyici Hatası CS1583
'dosya' geçerli bir Win32 kaynak dosyası değil  
  
 Bir dosya adı ile belirttiğinizde, bu hata oluşur. [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) geçerli veya doğru biçimlendirilmiş bir Win32 kaynak dosyası değil derleyici seçeneği. Visual Studio'da, Proje Tasarımcısı'nın uygulama bölmesinde dosya adı belirtildi.
