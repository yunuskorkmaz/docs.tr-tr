<span data-ttu-id="2aa65-103">'Name' yöntemi genel yöntem değil.</span><span class="sxs-lookup"><span data-stu-id="2aa65-103">The method 'name' is not a generic method.</span></span> <span data-ttu-id="2aa65-104">Bir ifade listesinde amaçlıyorsanız parantez içine alın < ifade.</span><span class="sxs-lookup"><span data-stu-id="2aa65-104">If you intended an expression list, use parentheses around the < expression.</span></span>

## <a name="example"></a><span data-ttu-id="2aa65-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="2aa65-106">Example</span></span>  
 <span data-ttu-id="2aa65-107">Aşağıdaki örnek CS0471 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="2aa65-107">The following example generates CS0471.</span></span>  
  
```  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
