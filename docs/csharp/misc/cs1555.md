# <a name="compiler-error-cs1555"></a>Derleyici hatası CS1555

Main yöntemi için belirtilen ' class ' bulunamadı  
  
 [**MainEntryPoint**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) derleyici seçeneğine bir sınıf belirtildi, ancak kaynak kodunda sınıf adı bulunamadı.
