 <span data-ttu-id="c7b84-105">Birçok yaygın hatalar, bu uyarıyı oluşturabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="c7b84-105">Several common mistakes can generate this warning:</span></span>
