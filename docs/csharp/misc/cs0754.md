## <a name="example"></a><span data-ttu-id="8a304-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="8a304-107">Example</span></span>  
 <span data-ttu-id="8a304-108">Aşağıdaki kod CS0754 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8a304-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
