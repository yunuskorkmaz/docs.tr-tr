# <a name="compiler-warning-level-1-cs3011"></a>Derleyici Uyarısı (düzey 1) CS3011
'member': yalnızca CLS uyumlu üyeler soyut olabilir  
  
 Bir sınıf üyesinin her ikisi birden olamaz [soyut](../../csharp/language-reference/keywords/abstract.md) ve uyumlu olmayan ortak dil belirtimi (CLS) ile. Sınıf üyelerinin tamamı uygulanması CLS belirtir. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [dil bağımsızlığı ve dilden bağımsız bileşenler](~/docs/standard/language-independence.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS3011 oluşturur:  
  
```csharp  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
