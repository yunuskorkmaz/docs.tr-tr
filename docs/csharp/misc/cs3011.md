## <a name="example"></a><span data-ttu-id="e3ffd-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="e3ffd-107">Example</span></span>  
 <span data-ttu-id="e3ffd-108">Aşağıdaki örnek CS3011 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e3ffd-108">The following example generates CS3011:</span></span>  
  
```  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
