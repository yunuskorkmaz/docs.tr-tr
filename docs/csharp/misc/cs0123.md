# <a name="compiler-error-cs0123"></a><span data-ttu-id="e3f31-103">Derleyici hatası CS0123</span><span class="sxs-lookup"><span data-stu-id="e3f31-103">Compiler Error CS0123</span></span>
<span data-ttu-id="e3f31-104">' Method ' için hiçbir aşırı yükleme ' Delegate ' temsilcisiyle eşleşmiyor</span><span class="sxs-lookup"><span data-stu-id="e3f31-104">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="e3f31-105">Doğru imza kullanılmadığından, temsilci oluşturma girişimi başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="e3f31-105">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="e3f31-106">Temsilcinin örnekleri, temsilci bildirimiyle aynı imzayla bildirilmelidir.</span><span class="sxs-lookup"><span data-stu-id="e3f31-106">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="e3f31-107">Yöntemi veya temsilci imzasını ayarlayarak bu hatayı çözebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="e3f31-107">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="e3f31-108">Daha fazla bilgi için bkz. [Temsilciler](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="e3f31-108">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="e3f31-109">Aşağıdaki örnek CS0123 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="e3f31-109">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
