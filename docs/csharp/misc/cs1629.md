 <span data-ttu-id="795e8-104">C# dil belirtimi yineleyicilerde güvenli olmayan koda izin vermez.</span><span class="sxs-lookup"><span data-stu-id="795e8-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="795e8-105">Aşağıdaki örnek CS1629 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="795e8-105">The following sample generates CS1629:</span></span>

// CS1629.cs

class C
{  
   IEnumerator<int> IteratorMeth() {

   }  
}
