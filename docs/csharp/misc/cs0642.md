 <span data-ttu-id="339d7-107">Aşağıdaki örnek CS0642 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="339d7-107">The following sample generates CS0642:</span></span>

```csharp  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
