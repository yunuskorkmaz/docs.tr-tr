# <a name="compiler-error-cs1649"></a><span data-ttu-id="1ba85-102">Derleyici Hatası CS1649</span><span class="sxs-lookup"><span data-stu-id="1ba85-102">Compiler Error CS1649</span></span>
<span data-ttu-id="1ba85-103">Salt okunur alan 'identifier' üyeleri ref geçirilemez veya çıkış (oluşturucu dışında)</span><span class="sxs-lookup"><span data-stu-id="1ba85-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="1ba85-104">Bir değişken bir üyesi olan bir işlev için geçirirseniz, bu hata oluşur bir `readonly` olarak alan bir `ref` veya `out` bağımsız değişken.</span><span class="sxs-lookup"><span data-stu-id="1ba85-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="1ba85-105">Bu yana `ref` ve `out` parametreleri işlevi tarafından değiştirilebilir, bu verilmez.</span><span class="sxs-lookup"><span data-stu-id="1ba85-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="1ba85-106">Bu hatayı gidermek için kaldırma `readonly` alan anahtar sözcüğü veya üyeleri geçmeyin `readonly` işlev alanı.</span><span class="sxs-lookup"><span data-stu-id="1ba85-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="1ba85-107">Örneğin, deneyebilecekleriniz değiştirilebilir geçici bir değişken oluşturma ve geçici olarak geçirerek bir `ref` bağımsız değişken, aşağıdaki örnekte gösterildiği gibi.</span><span class="sxs-lookup"><span data-stu-id="1ba85-107">For example, you might try creating a temporary variable which can be modified and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1ba85-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="1ba85-108">Example</span></span>  
 <span data-ttu-id="1ba85-109">Aşağıdaki örnek, CS1649 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1ba85-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
