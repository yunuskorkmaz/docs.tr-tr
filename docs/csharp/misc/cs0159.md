# <a name="compiler-error-cs0159"></a>Derleyici hatası CS0159
Goto ifadesinin kapsamı içinde ' Label ' etiketi yok  
  
 [Goto](../language-reference/keywords/goto.md) ifadesinin başvurduğu etiket deyimin kapsamı içinde bulunamadı `goto` .  
  
 Aşağıdaki örnek CS0159 oluşturur:  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
