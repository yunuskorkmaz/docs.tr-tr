# <a name="compiler-error-cs0457"></a><span data-ttu-id="edd3d-102">Derleyici Hatası CS0457</span><span class="sxs-lookup"><span data-stu-id="edd3d-102">Compiler Error CS0457</span></span>

 <span data-ttu-id="edd3d-105">Bu hataya neden olabilecek bir senaryo aşağıdaki gibidir:</span><span class="sxs-lookup"><span data-stu-id="edd3d-105">One scenario that can cause this error is as follows:</span></span>

 <span data-ttu-id="edd3d-109">Derleyici, ilk dönüşüm en iyi hedef türünü, ikinci dönüşüm ise en iyi kaynak türünü sağladığından, iki dönüştürme yöntemini eşit olarak ağırlıka getirir.</span><span class="sxs-lookup"><span data-stu-id="edd3d-109">The compiler will weight the two conversion methods equally, because the first conversion provides the best destination type, and the second conversion provides the best source type.</span></span> <span data-ttu-id="edd3d-110">Derleyici seçemeyeceğinden, bu hata oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="edd3d-110">Since the compiler will be unable to choose, this error will be generated.</span></span> <span data-ttu-id="edd3d-111">Çözmek için, A'yı B'ye dönüştüren yeni bir açık yöntem yazın.</span><span class="sxs-lookup"><span data-stu-id="edd3d-111">To resolve, write a new explicit method converting A to B.</span></span>  
  
 <span data-ttu-id="edd3d-112">Bu hataya neden olan başka bir senaryo, A'yı B'ye dönüştüren iki yöntem olmasıdır. ## <a name="example"></a><span data-ttu-id="edd3d-113">Örnek</span><span class="sxs-lookup"><span data-stu-id="edd3d-113">Example</span></span>  
 <span data-ttu-id="edd3d-114">Aşağıdaki örnek CS0457 üretir.</span><span class="sxs-lookup"><span data-stu-id="edd3d-114">The following sample generates CS0457.</span></span>  
  
```csharp  
// CS0457.cs  
using System;  
public class A { }  
  
public class G0 {  }  
public class G1<R> : G0 {  }  
  
public class H0 {  
   public static implicit operator G0(H0 h) {  
      return new G0();  
   }  
}  
public class H1<R> : H0 {  
   public static implicit operator G1<R>(H1<R> h) {  
      return new G1<R>();  
   }  
}  
  
public class Test
{  
   public static void F0(G0 g) {  }  
   public static void Main()
   {  
      H1<A> h1a = new H1<A>();  
      F0(h1a);   // CS0457  
   }  
}  
```
