# <a name="compiler-error-cs0631"></a>Derleyici Hatası CS0631
ref ve out bu bağlamda geçerli değil  
  
 Bildirimi bir [dizin oluşturucu](../../csharp/programming-guide/indexers/index.md) kullanımını içeremez [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out.md) parametreleri.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0631 oluşturur:  
  
```  
// CS0631.cs  
public class MyClass  
{  
    public int this[ref int i]   // CS0631  
    // try the following line instead  
    // public int this[int i]  
    {  
        get  
        {  
            return 0;  
        }  
    }  
}  
  
public class MyClass2  
{  
    public static void Main()  
    {  
    }  
}  
```
