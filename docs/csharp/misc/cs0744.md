# <a name="compiler-error-cs0744"></a><span data-ttu-id="cfa9b-102">Derleyici Hatası CS0744</span><span class="sxs-lookup"><span data-stu-id="cfa9b-102">Compiler Error CS0744</span></span>
<span data-ttu-id="cfa9b-103">Beklenen bağlamsal anahtar sözcük, 'equals'</span><span class="sxs-lookup"><span data-stu-id="cfa9b-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="cfa9b-104">Desenini bir `join` yan tümcesi `join`... `in`... `on`... `equals`, bu örnekte gösterildiği gibi:</span><span class="sxs-lookup"><span data-stu-id="cfa9b-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y  
            select x;

## <a name="example"></a><span data-ttu-id="cfa9b-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="cfa9b-107">Example</span></span>  
 <span data-ttu-id="cfa9b-108">Aşağıdaki kod CS0744 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cfa9b-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
