## <a name="example"></a><span data-ttu-id="11291-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="11291-108">Example</span></span>  
 <span data-ttu-id="11291-109">Dosyanın ikinci bir sınıf tanımlayan bir DLL oluşturur `C` sınıfından türetilen `B` önceki örnekte oluşturuldu.</span><span class="sxs-lookup"><span data-stu-id="11291-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

  
## <a name="example"></a><span data-ttu-id="11291-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="11291-110">Example</span></span>
