# <a name="compiler-error-cs0765"></a><span data-ttu-id="ac4a1-102">Derleyici Hatası CS0765</span><span class="sxs-lookup"><span data-stu-id="ac4a1-102">Compiler Error CS0765</span></span>
<span data-ttu-id="ac4a1-103">Yalnızca bir tanımlama bildirimi olan kısmi yöntemler veya kaldırılmış koşullu yöntemler ifade ağaçlarında kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="ac4a1-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="ac4a1-104">Kaldırılan bir kısmi yöntem çağrısı bir ifade olsa da, kabul edilebilir bir deyim bir ifade ağacı içinde değil.</span><span class="sxs-lookup"><span data-stu-id="ac4a1-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ac4a1-105">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="ac4a1-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="ac4a1-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="ac4a1-107">Example</span></span>  
 <span data-ttu-id="ac4a1-108">Aşağıdaki kod, iki konumda CS0765 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ac4a1-108">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
