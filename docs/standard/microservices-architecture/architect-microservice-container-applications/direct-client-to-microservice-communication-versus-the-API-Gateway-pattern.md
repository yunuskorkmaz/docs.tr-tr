## <a name="drawbacks-of-the-api-gateway-pattern"></a><span data-ttu-id="74adb-243">API ağ geçidi desenini dezavantajları</span><span class="sxs-lookup"><span data-stu-id="74adb-243">Drawbacks of the API Gateway pattern</span></span>
