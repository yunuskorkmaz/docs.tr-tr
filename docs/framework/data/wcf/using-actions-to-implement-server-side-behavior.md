 <span data-ttu-id="7be98-158">Parametreler belirteç olarak geçirilebilir.</span><span class="sxs-lookup"><span data-stu-id="7be98-158">The parameters may be passed as tokens.</span></span> <span data-ttu-id="7be98-159">Bunun nedeni, kaynakları temsil eden belirteçlerle birlikte çalışacak bir veri hizmeti sağlayıcısı yazmak, bu durum gerçek eyleme göndermeden önce bu belirteçleri gerçek kaynaklara dönüştürmeniz (sıralamalarınızda) gerekebilir.</span><span class="sxs-lookup"><span data-stu-id="7be98-159">This is because it is possible to write a Data Service Provider that works with tokens that represent resources, if this is the case you may need to convert (marshal) these tokens into actual resources before dispatching to the actual action.</span></span> <span data-ttu-id="7be98-160">Parametre sıralandıktan sonra, kaynak, eylem çağrıldığında gerçekleşen tüm değişikliklerin diske kaydedilip diske yazılacağı şekilde düzenlenebilir bir durumda olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="7be98-160">After the parameter has been marshaled, it must be in an editable state so that any changes to the resource that occur when the action is invoked will be saved and written to disk.</span></span>  
  
 <span data-ttu-id="7be98-161">Bu arabirim için iki yöntem gereklidir: Invoke ve GetResult.</span><span class="sxs-lookup"><span data-stu-id="7be98-161">This interface requires two methods: Invoke and GetResult.</span></span> <span data-ttu-id="7be98-162">Invoke komut, eylemin davranışını uygulayan temsilciyi çağırır ve GetResult eylemin sonucunu döndürür.</span><span class="sxs-lookup"><span data-stu-id="7be98-162">Invoke invokes the delegate that implements the action’s behavior and GetResult returns the result of the action.</span></span>  
  
## <a name="invoking-a-wcf-data-service-action"></a><span data-ttu-id="7be98-163">WCF veri hizmeti eylemini çağırma</span><span class="sxs-lookup"><span data-stu-id="7be98-163">Invoking a WCF Data Service Action</span></span>  
 <span data-ttu-id="7be98-164">Eylemler bir HTTP POST isteği kullanılarak çağrılır.</span><span class="sxs-lookup"><span data-stu-id="7be98-164">Actions are invoked using an HTTP POST request.</span></span> <span data-ttu-id="7be98-165">URL, kaynağı ve ardından eylem adını belirtir.</span><span class="sxs-lookup"><span data-stu-id="7be98-165">The URL specifies the resource followed by the action name.</span></span> <span data-ttu-id="7be98-166">Parametreler isteğin gövdesine geçirilir.</span><span class="sxs-lookup"><span data-stu-id="7be98-166">Parameters are passed in the body of the request.</span></span> <span data-ttu-id="7be98-167">Örneğin, MovieService adlı bir hizmet varsa, oran adlı bir eylem ortaya çıkarılır.</span><span class="sxs-lookup"><span data-stu-id="7be98-167">For example if there was a service called MovieService which exposed an action called Rate.</span></span> <span data-ttu-id="7be98-168">Belirli bir filmin hız eylemini çağırmak için aşağıdaki URL 'YI kullanabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="7be98-168">You could use the following URL to invoke the Rate action on a specific movie:</span></span>  
  
 `http://MovieServer/MovieService.svc/Movies(1)/Rate`
  
 <span data-ttu-id="7be98-169">Filmler (1) derecelendirmek istediğiniz filmi belirtir ve oran ücret eylemini belirtir.</span><span class="sxs-lookup"><span data-stu-id="7be98-169">Movies(1) specifies the movie you wish to rate and Rate specifies the Rate action.</span></span> <span data-ttu-id="7be98-170">Derecelendirmenin gerçek değeri, aşağıdaki örnekte gösterildiği gibi HTTP isteğinin gövdesinde olacaktır:</span><span class="sxs-lookup"><span data-stu-id="7be98-170">The actual value of the rating will be in the body of the HTTP request as shown in the following example:</span></span>

Content-Type: application/json   
Content-Length: 20   
Host: localhost:15238

   "rating": 4

> [!WARNING]
> <span data-ttu-id="7be98-171">Yukarıdaki örnek kod yalnızca JSON ışığı desteği olan WCF Veri Hizmetleri 5,2 ve üzeri sürümlerle çalışır.</span><span class="sxs-lookup"><span data-stu-id="7be98-171">The above sample code will work only with WCF Data Services 5.2 and later which has support for JSON light.</span></span> <span data-ttu-id="7be98-172">WCF Veri Hizmetleri önceki bir sürümünü kullanıyorsanız, JSON ayrıntılı içerik türü olarak şu şekilde belirtmeniz gerekir: `application/json;odata=verbose`.</span><span class="sxs-lookup"><span data-stu-id="7be98-172">If using an earlier version of WCF Data Services, you must specify the json verbose content-type as follows: `application/json;odata=verbose`.</span></span>

 <span data-ttu-id="7be98-174">Yukarıdaki kod parçacığında, `MoviesModel` sınıfı bir WCF veri hizmetine hizmet başvurusu Ekle için Visual Studio kullanılarak oluşturulmuştur.</span><span class="sxs-lookup"><span data-stu-id="7be98-174">In the code snippet above, the `MoviesModel` class was generated by using Visual Studio to Add Service Reference to a WCF Data Service.</span></span>
