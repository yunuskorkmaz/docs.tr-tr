 <span data-ttu-id="02a02-158">Parametre belirteçleri geçirilebilir.</span><span class="sxs-lookup"><span data-stu-id="02a02-158">The parameters may be passed as tokens.</span></span> <span data-ttu-id="02a02-159">Veri hizmeti sağlayıcısı (hazırlama) dönüştürmeniz gerekebilir Durum buysa, kaynakları temsil eden belirteçleri ile çalışan gerçek eyleme dağıtmadan önce bu belirteçleri gerçek Kaynakları Yazmayı mümkün olduğundan bu değildir.</span><span class="sxs-lookup"><span data-stu-id="02a02-159">This is because it is possible to write a Data Service Provider that works with tokens that represent resources, if this is the case you may need to convert (marshal) these tokens into actual resources before dispatching to the actual action.</span></span> <span data-ttu-id="02a02-160">Parametre sıraya sonra böylelikle eylemi çağrıldığında oluşan değişiklikleri kaynağa kaydedilmiş ve üzerine yazılma disk düzenlenebilir bir durumda olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="02a02-160">After the parameter has been marshalled, it must be in an editable state so that any changes to the resource that occur when the action is invoked will be saved and written to disk.</span></span>

## <a name="invoking-a-wcf-data-service-action"></a><span data-ttu-id="02a02-163">Bir WCF veri hizmeti eylemi çağırma</span><span class="sxs-lookup"><span data-stu-id="02a02-163">Invoking a WCF Data Service Action</span></span>  
 <span data-ttu-id="02a02-164">Eylemler, bir HTTP POST isteği kullanılarak çağrılır.</span><span class="sxs-lookup"><span data-stu-id="02a02-164">Actions are invoked using an HTTP POST request.</span></span> <span data-ttu-id="02a02-165">Ardından eylem adı kaynak URL'sini belirtir.</span><span class="sxs-lookup"><span data-stu-id="02a02-165">The URL specifies the resource followed by the action name.</span></span> <span data-ttu-id="02a02-166">Parametreleri istek gövdesinde geçirilir.</span><span class="sxs-lookup"><span data-stu-id="02a02-166">Parameters are passed in the body of the request.</span></span> <span data-ttu-id="02a02-167">Örneğin, varsa oranı olarak adlandırılan bir eylem ifşa eden MovieService adlı bir hizmet.</span><span class="sxs-lookup"><span data-stu-id="02a02-167">For example if there was a service called MovieService which exposed an action called Rate.</span></span> <span data-ttu-id="02a02-168">Belirli bir filmi oranı eylemi çağırmak için aşağıdaki URL'yi kullanabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="02a02-168">You could use the following URL to invoke the Rate action on a specific movie:</span></span>  
  
 `http://MovieServer/MovieService.svc/Movies(1)/Rate`
  
 <span data-ttu-id="02a02-169">Movies(1) hızı ve hızı için istediğiniz film oranı eylemi belirtir.</span><span class="sxs-lookup"><span data-stu-id="02a02-169">Movies(1) specifies the movie you wish to rate and Rate specifies the Rate action.</span></span> <span data-ttu-id="02a02-170">Derecelendirme gerçek değerini HTTP istek gövdesinde, aşağıdaki örnekte gösterildiği gibi olacaktır:</span><span class="sxs-lookup"><span data-stu-id="02a02-170">The actual value of the rating will be in the body of the HTTP request as shown in the following example:</span></span>  
  
```  
POST http://MovieServer/MoviesService.svc/Movies(1)/Rate HTTP/1.1   
Content-Type: application/json   
Content-Length: 20   
Host: localhost:15238  
{   
   "rating": 4   
}  
```  
  
> [!WARNING]
> <span data-ttu-id="02a02-171">Yukarıdaki örnek kod, yalnızca WCF Veri Hizmetleri 5.2 ile çalışır ve daha sonra JSON Light için desteği vardır.</span><span class="sxs-lookup"><span data-stu-id="02a02-171">The above sample code will work only with WCF Data Services 5.2 and later which has support for JSON light.</span></span> <span data-ttu-id="02a02-172">WCF Data Services'ın önceki bir sürümünü kullanıyorsanız, json verbose içerik türü şu şekilde belirtmeniz gerekir: `application/json;odata=verbose`.</span><span class="sxs-lookup"><span data-stu-id="02a02-172">If using an earlier version of WCF Data Services, you must specify the json verbose content-type as follows: `application/json;odata=verbose`.</span></span>

//...

 <span data-ttu-id="02a02-174">Yukarıdaki kod parçacığında `MoviesModel` sınıfı, WCF veri hizmetine hizmet başvurusu eklemek için Visual Studio kullanılarak oluşturuldu.</span><span class="sxs-lookup"><span data-stu-id="02a02-174">In the code snippet above, the `MoviesModel` class was generated by using Visual Studio to Add Service Reference to a WCF Data Service.</span></span>
