 <span data-ttu-id="bd8a7-108">Bir LINQ Sorgu işlemi üç eylemden oluşur: sorguyu veri kaynağından veya kaynaklarından elde etmek ve sorgu oluşturun.</span><span class="sxs-lookup"><span data-stu-id="bd8a7-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>
