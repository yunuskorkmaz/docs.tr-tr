# <a name="dynamicresource-markup-extension"></a><span data-ttu-id="71ac3-102">DynamicResource Biçimlendirme Uzantısı</span><span class="sxs-lookup"><span data-stu-id="71ac3-102">DynamicResource Markup Extension</span></span>
<span data-ttu-id="71ac3-103">İçin herhangi bir değer sağlar [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] özellik özniteliği tarafından tanımlanan kaynağına başvuru olması için bu değeri erteleniyor.</span><span class="sxs-lookup"><span data-stu-id="71ac3-103">Provides a value for any [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] property attribute by deferring that value to be a reference to a defined resource.</span></span> <span data-ttu-id="71ac3-104">Bu kaynak için arama davranışı için çalışma zamanı arama benzerdir.</span><span class="sxs-lookup"><span data-stu-id="71ac3-104">Lookup behavior for that resource is analogous to run-time lookup.</span></span>  
  
## <a name="xaml-attribute-usage"></a><span data-ttu-id="71ac3-105">XAML Öznitelik Kullanımı</span><span class="sxs-lookup"><span data-stu-id="71ac3-105">XAML Attribute Usage</span></span>  
  
```xml  
<object property="{DynamicResource key}" .../>  
```

```xml  
  </object.property>  
</object>  
```
