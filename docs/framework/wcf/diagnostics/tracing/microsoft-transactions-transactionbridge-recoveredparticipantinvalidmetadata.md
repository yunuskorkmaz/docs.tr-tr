# <a name="microsofttransactionstransactionbridgerecoveredparticipantinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
Katılımcı kurtarma günlüğü girişinde bulunan bitiş noktası başvurusu geçersiz ya da uyumsuz meta veri içeriyor.  
  
## <a name="description"></a>Açıklama  
 Sistem kurtarılmış katılımcı listesi için bir proxy oluşturamıyor.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Sorun devam ederse Microsoft ile iletişim kurun.
