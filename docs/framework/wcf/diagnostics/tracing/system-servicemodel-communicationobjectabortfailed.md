# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
İletişim nesnesi iptal edilemedi.  
  
## <a name="description"></a>Açıklama  
 İletişim nesnesi Abort yöntemini yürütülmesi sırasında özel durum oluştu olduğundan iptal edilemedi. İzleme yakındaki başka bir özel durumla ilgili ayrıntılar içerebilir.
