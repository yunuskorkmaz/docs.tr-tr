## <a name="syntax"></a>Sözdizimi  
  
```  
void CloseEnum (  
    [in] HCORENUM     hEnum  
);  
```  
  
#### <a name="parameters"></a>Parametreler

 [in] Kapatılması numaralandırması örneği.
