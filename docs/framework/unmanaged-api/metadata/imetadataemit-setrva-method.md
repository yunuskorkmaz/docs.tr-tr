# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA Yöntemi
Belirtilen yöntem göreli sanal adresini ayarlar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,   
    [in]  ULONG        ulRVA   
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `md`  
 [in] Hedef yöntemi veya yöntem uygulaması için belirteci.  
  
 `ulRVA`  
 [in] Kod veya veri alanı adresi.
