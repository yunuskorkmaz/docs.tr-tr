# <a name="persistence"></a>Kalıcılığı
Bu bölümde kalıcı iş akışları dayanıklı Orta Windows Workflow Foundation (WF) gösteren örnekleri içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [İş Akışı Uygulamasını Kalıcı Yapma](../../../../docs/framework/windows-workflow-foundation/samples/persisting-a-workflow-application.md)  
 Nasıl çalıştırılacağını gösteren bir <xref:System.Activities.WorkflowApplication>boşta gittiğinde kaldırın ve sonra yürütülmeye devam etmek için yeniden yükleyin.  
  
 [Yerleşik Yapılandırma](../../../../docs/framework/windows-workflow-foundation/samples/built-in-configuration.md)  
 Kullanım ve SQL iş akışı örneği depolama yapılandırmasını gösterir.  
  
 [SQLStoreExtensibility](../../../../docs/framework/windows-workflow-foundation/samples/sqlstoreextensibility.md)  
 Kullanım ve SQL iş akışı örneği deposundaki yükseltilen özellikleri yapılandırmasını gösterir.  
  
 [Özellik Yükseltme Etkinliği](../../../../docs/framework/windows-workflow-foundation/samples/property-promotion-activity.md)  
 Bu örnek tümleşen bir uçtan uca çözümü sağlar <xref:System.Activities.DurableInstancing.SqlWorkflowInstanceStore> iş akışı yazma deneyimi doğrudan yükseltme özelliği.
