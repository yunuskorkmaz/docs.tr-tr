## <a name="virtual"></a><span data-ttu-id="22855-129">Sanal</span><span class="sxs-lookup"><span data-stu-id="22855-129">Virtual</span></span>

 <span data-ttu-id="22855-133">Eklediğiniz <xref:System.Windows.Forms.DataGridView> yönettiğiniz bir önbellek denetimine ve veri satırı gönderilir ve çekilen kod denetimleri.</span><span class="sxs-lookup"><span data-stu-id="22855-133">You attach the <xref:System.Windows.Forms.DataGridView> control to a cache that you manage, and your code controls when data rows are pushed and pulled.</span></span> <span data-ttu-id="22855-134">Bellek alanını küçük tutmak için önbellek boyutu şu anda görüntülenen satır sayısını benzer olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="22855-134">To keep the memory footprint small, the cache should be similar in size to the number of rows currently displayed.</span></span> <span data-ttu-id="22855-135">Kullanıcı yeni satırlar görünüme kaydırdığında kodunuzu yeni verileri önbellekten ister ve isteğe bağlı olarak bellekten eski verileri temizler.</span><span class="sxs-lookup"><span data-stu-id="22855-135">When the user scrolls new rows into view, your code requests new data from the cache and optionally flushes old data from memory.</span></span>
