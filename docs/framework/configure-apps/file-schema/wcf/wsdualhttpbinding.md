## <a name="remarks"></a><span data-ttu-id="b4b51-200">Açıklamalar</span><span class="sxs-lookup"><span data-stu-id="b4b51-200">Remarks</span></span>  
 <span data-ttu-id="b4b51-201">`WSDualHttpBinding` Web hizmeti protokolleri aynı destek sağlayan `WSHttpBinding`, ancak çift yönlü sözleşmeler ile kullanım için.</span><span class="sxs-lookup"><span data-stu-id="b4b51-201">The `WSDualHttpBinding` provides the same support for Web Service protocols as the `WSHttpBinding`, but for use with duplex contracts.</span></span> <span data-ttu-id="b4b51-202">`WSDualHttpBinding` yalnızca SOAP Güvenliği destekler ve güvenilir Mesajlaşma gerektirir.</span><span class="sxs-lookup"><span data-stu-id="b4b51-202">`WSDualHttpBinding` only supports SOAP security and requires reliable messaging.</span></span> <span data-ttu-id="b4b51-203">Bu bağlama istemci hizmeti için bir geri çağırma uç noktası sağlayan ortak bir URI sahip olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="b4b51-203">This binding requires that the client has a public URI that provides a callback endpoint for the service.</span></span> <span data-ttu-id="b4b51-204">Bu tarafından sağlanan `clientBaseAddress` özniteliği.</span><span class="sxs-lookup"><span data-stu-id="b4b51-204">This is provided by the `clientBaseAddress` attribute.</span></span> <span data-ttu-id="b4b51-205">Bir çift bağlama hizmeti istemcinin IP adresini gösterir.</span><span class="sxs-lookup"><span data-stu-id="b4b51-205">A dual binding exposes the IP address of the client to the service.</span></span> <span data-ttu-id="b4b51-206">İstemci, yalnızca Hizmetleri güvenleri bağladığı emin olmak için güvenlik kullanmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="b4b51-206">The client should use security to ensure that it only connects to services it trusts.</span></span>  
  
 <span data-ttu-id="b4b51-207">Bu bağlama, güvenilir bir şekilde bir veya daha fazla SOAP aracılar iletişim kurmak için kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="b4b51-207">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>  
  
 <span data-ttu-id="b4b51-208">Varsayılan olarak, WS-ReliableMessaging çalışma zamanı yığın güvenilirlik, WS-güvenlik ileti güvenliği ve kimlik doğrulama, ileti teslimi, HTTP için bu bağlama oluşturur ve bir Text/XML ileti kodlama.</span><span class="sxs-lookup"><span data-stu-id="b4b51-208">By default, this binding generates a runtime stack with WS-ReliableMessaging for reliability, WS-Security for message security and authentication, HTTP for message delivery, and a Text/XML message encoding.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b4b51-209">Örnek</span><span class="sxs-lookup"><span data-stu-id="b4b51-209">Example</span></span>  
  
```xml  
<configuration>  
<system.ServiceModel>  
<bindings>  
<wsDualHttpBinding>  
    <binding   
        closeTimeout="00:00:10"  
        openTimeout="00:00:20"   
        receiveTimeout="00:00:30"  
        sendTimeout="00:00:40"

</bindings>  
</system.ServiceModel>  
</configuration>
```  
