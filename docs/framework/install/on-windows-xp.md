# <a name="install-the-net-framework-on-windows-xp-and-windows-server-2003"></a>.NET Framework, Windows XP ve Windows Server 2003 üzerinde yükleme

> [!NOTE]
> Windows XP artık Microsoft tarafından desteklenir. Windows desteklenir ve .NET Framework'ün en son sürümünü içerir, 10, yükseltmeniz önerilir. Bu belge, yalnızca bir yararlı sorun giderme kılavuzu sağlanır.

.NET Framework, Windows üzerinde birçok uygulama çalıştırmak için gereklidir. Yüklemek için aşağıdaki yönergeleri kullanabilirsiniz. Bu sayfada bir uygulama çalıştırmaya ve makinenizde aşağıdaki iletişim kutusu görmesini sonra gelmiş.

![Bu uygulama başlatılamadı](./media/this-application-could-not-be-started.png)

Bu yönergeler ihtiyacınız .NET Framework sürümlerini yüklemenize yardımcı olur. [.NET Framework 4.7.2](https://go.microsoft.com/fwlink/?LinkID=863255) en son sürüm. Windows XP ve Windows Server 2003'te desteklenmiyor. İle birlikte gelir [Windows 10 Nisan 2018 güncelleştirmesi](https://www.microsoft.com/software-download/windows10).

## <a name="net-framework-403"></a>.NET framework 4.0.3

[.NET Framework 4.0.3](https://go.microsoft.com/fwlink/?LinkID=213834) en son desteklenen .NET Framework sürümü Windows XP ve Windows Server 2003 üzerinde. .NET Framework 4.0.3 gerektiren [.NET Framework 4](https://go.microsoft.com/fwlink/?LinkID=213834) ilk olarak yüklenir. Bu .NET Framework sürümlerinin her ikisi de, artık Microsoft tarafından desteklenir.

## <a name="net-framework-4"></a>.NET Framework 4

Yükleyebileceğiniz [.NET Framework 4](https://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) Windows XP. Artık Microsoft tarafından desteklenir.

## <a name="net-framework-35"></a>.NET Framework 3.5

Yükleyebileceğiniz [.NET Framework 3.5](https://go.microsoft.com/fwlink/?LinkID=213834&dotnetdocs) Windows XP.

.NET Framework 3.5, .NET Framework 1.0 için 3.5 oluşturulan uygulamalarını çalıştırmak için kullanılabilir.
